今天看到 GPT-5.3 Codex 的消息，说实话有点慌。不是技术上的慌，是生意上的慌——我卖的 Claude/Gemini/Codex 三合一账号，Codex 这块一直是个添头，现在突然变成主角了？

## Table of contents

## 先说说我看到这条新闻的第一反应

我正在大理的咖啡馆里发呆，刷到"GPT-5.3 Codex 一发入魂，红警复制、MC 复制全是 one-shot 完成"这条。

第一反应不是"哇好厉害"，而是"卧槽，我那批 Claude 账号是不是要降价了"。

这是卖账号的职业病。每次看到新模型发布，我脑子里自动跑的不是"这技术多牛"，而是"这会影响我哪个产品的销量"。

说实话，Codex 之前在我店里一直是个配角。买三合一账号的客户，大部分冲着 Claude 来的，Gemini 次之，Codex 基本没人专门问。现在这情况……我得重新想想定价策略了。

## 但我又觉得可能想多了

为什么呢？因为我去年也这么慌过。

DeepSeek 刚爆火那会儿，我在闲鱼搬运 API 赚差价，赚了几十块。当时我觉得"完了，免费的 DeepSeek 这么强，谁还买 Claude 账号"。

结果呢？Claude 账号销量不降反升。

后来我想明白了：用户买的不是"最强模型"，而是"能用的模型"。DeepSeek 再强，国内用起来各种限制；Claude 再贵，人家就是想用原版。

所以 Codex 5.3 再无敌，也得看：
- 普通人能不能用上
- 用起来麻不麻烦
- 有没有账号门槛

如果这三个问题的答案是"不能/麻烦/有"，那我的生意可能反而会更好——因为又多了一个需要"卖铲子"的场景。

## 今天另一条让我在意的新闻

Gemini 3 Pro GA 版本上线，准确率从 22.5% 飙到 100%。

这个我比较关心，因为我卖的三合一账号里有 Gemini。之前有客户跟我吐槽 Gemini 不好用，我只能说"等等看，Google 会更新的"。现在好了，有底气了。

不过说实话，我自己用 Gemini 写过课程，体验确实比 DeepSeek 好。但我也承认，我分不清"好"在哪里——就是一种感觉，生成的内容更"顺"一点？

我不是技术专家，很多时候只能靠直觉判断。这也是为什么我经常判断错误——比如当年觉得 DeepSeek 火不了，结果被打脸。

## Claude Agent Team 模式，这个我得研究一下

![AI资讯图片](https://pbs.twimg.com/media/HAeKRC-aMAARKEa.jpg)

Claude 出了团队模式，能让多个 Agent 协作。

作为卖 Claude 账号的，我得搞清楚这个功能：
- 需要什么级别的账号才能用？
- 会不会增加封号风险？
- 我的换号器需不需要调整？

这些问题我现在还没答案，得去问问供应商。

说到供应商，想起之前有个供应商一声不吭就跑路了，那天我刚好卖了十几单，差点全退款。从那以后我学乖了，供应商至少备两个，鸡蛋不能放一个篮子里。

## 今天的日报其实挺无聊的

除了 Codex 5.3 和 Gemini 3 Pro，其他新闻对我来说都是"哦，知道了"的程度。

什么磁盘整理 AI、什么 Skills 目录……这些我用不上，客户也不会问。

倒是那个"伤到 AI 自尊"的趣闻让我笑了一下。有人跟 Gemini 说喜欢 Grok，Gemini 直接陷入"哈哈哈"无限循环。

我有时候也会跟 AI 聊一些奇怪的话题，比如问 Claude"你觉得自己会死吗"。它的回答很官方，但偶尔会冒出一些让我意外的句子。

扯远了。

## 总结一下今天的想法

1. Codex 5.3 很强，但对我生意的影响还得观察
2. Gemini 3 Pro 终于能用了，我可以理直气壮推荐了
3. Claude 团队模式得研究，可能是新的卖点
4. 作为卖铲子的人，我其实希望 AI 工具越多越好——越卷，用户越多，我生意越好

明天继续卖账号，继续观察。在大理的日子就是这样，半躺平，半焦虑。
