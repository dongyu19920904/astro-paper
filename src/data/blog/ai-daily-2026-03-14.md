今天看到 Claude 把 100 万上下文开放给所有用户，第一反应不是"哇好厉害"，而是——我那批 Claude 账号会不会因为这个更好卖了？

## Table of contents

## 卖账号的人看AI新闻，角度就是不一样

做了一年多 AI 账号生意，我发现自己看新闻的视角已经彻底变了。普通用户看到"Claude 100万上下文不加价"会想"太好了可以处理更大的项目了"，我想的是"这会让多少人从 GPT 转过来？我要不要多进点货？"

Claude 这次更新确实狠。以前处理超大项目，要么砍上下文要么多花一倍钱，现在直接 100 万 token 全开放，超过 200K 也不收 2 倍价格了。单次能处理的图片和 PDF 从 100 页飙到 600 页，Opus 4.6 在 100 万上下文时的召回率 78.3%，直接碾压 GPT-5.4 的 36.6%。

这对我来说意味着什么？Claude 账号可能会更好卖。但同时也意味着——竞争会更激烈，因为大家都会盯上这块肥肉。

## 但今天真正让我紧张的是另一条新闻

Cursor 的两位核心负责人跳槽 xAI 了。工程负责人 Andrew Milich 和产品负责人 Jason Ginsberg，这两人是 Cursor 从零做到 20 亿美元 ARR 的核心操盘手。

我卖 Cursor 换号器已经卖了大半年了，这是我店里最稳定的产品之一。看到这条新闻的时候，我脑子里第一个念头是——xAI 要做编程产品了，而且很可能做得很好。

AI 编程现在是超过 50 亿美元的市场：Cursor 20 亿、Claude Code 25 亿、GitHub Copilot 超 10 亿。xAI 不缺算力（Memphis 的 Colossus 集群有 10 万张 H100），也有 Grok 3，缺的就是真正把 AI 编程产品从 0 做到 20 亿的产品团队。现在这块拼图补上了。

如果 xAI 的编程产品今年 Q2 就出来（我觉得概率挺大的），那我得提前想好怎么应对。是跟进卖 xAI 的账号？还是继续押注 Cursor 和 Claude？

## 我的判断经常错，但这次感觉不一样

说实话，我判断错过很多次。去年 DeepSeek 刚出来的时候，我觉得它火不了，结果它爆了。我在闲鱼搬运 DeepSeek API 赚了几十块，但也就那样。

但这次不一样。xAI 挖走 Cursor 的核心团队，这不是"可能会做"，而是"一定会做，而且会做得很好"。马斯克做事的风格大家都知道，要么不做，要做就做到极致。

我现在在想，如果 xAI 的编程产品真的出来了，会不会像当年 ChatGPT 一样，直接改变整个市场格局？Cursor 会不会因为核心团队出走而掉队？Claude Code 会不会因为 xAI 的竞争而降价？

这些问题我现在都没答案。但作为一个卖账号的，我得提前做好准备。

## 在大理卖账号，有时候挺焦虑的

我在大理旅居，每天的生活节奏很慢。早上起来泡杯咖啡，看看日报，处理一下订单，下午去洱海边发发呆。听起来很惬意，但其实心里一直有点焦虑。

AI 这个行业变化太快了。今天还是 Cursor 一家独大，明天可能就是 xAI 横空出世。我卖的账号，今天还是刚需，明天可能就被新产品替代了。

但话说回来，这也是这个行业的魅力所在。每天都有新东西出来，每天都有新机会。我不是技术大牛，也不是产品经理，但我能敏锐地感知到市场的变化，然后快速调整策略。

这可能就是"卖铲子的人"的生存之道吧。AI 淘金热里，我不挖矿，我卖工具。只要淘金热还在，我就有生意做。

## 最后说一句

今天的新闻让我有点紧张，但也有点兴奋。紧张是因为不确定性，兴奋是因为新机会。

如果你也在关注 AI 编程工具，或者也在做相关的生意，不妨多留意一下 xAI 的动向。我有种预感，今年 Q2 可能会是 AI 编程赛道的一个转折点。

至于我，先把手头的 Claude 账号和 Cursor 换号器卖好，然后等着看 xAI 会搞出什么幺蛾子。
