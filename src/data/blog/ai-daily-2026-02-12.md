今天刷日报的时候看到 DeepSeek 悄悄灰度新模型，知识库更新到 2025 年 5 月，上下文窗口拉到 100 万。说实话，我第一反应不是"哇好厉害"，而是——我得去问问供应商，API 价格会不会变。

## Table of contents

## 卖账号的人看新闻，视角就是不一样

这大概是我做这行养成的职业病。别人看 AI 新闻想的是"这技术牛不牛"，我想的是"这会影响我哪个产品"。

DeepSeek 这次更新挺有意思的，不是大张旗鼓发布会那种，而是"偷偷"灰度。你得先把 APP 升级到 1.7.4，在手机上聊一句，再回网页版才能触发。这操作让我想起去年 DeepSeek 刚火的时候，我在闲鱼搬运 API 赚差价，那会儿信息差就是钱。现在这种"偷偷更新"的玩法，又是一波信息差——知道的人已经在体验 100 万上下文了，不知道的还在用老版本。

不过说实话，我当年判断 DeepSeek 火不了，结果被打脸打得啪啪响。所以现在我对自己的判断都持保留态度。

## GLM-5 这波操作，让我想起被闲鱼封号的日子

智谱这次调价引发的用户不满，我太能理解了。

以前 Pro 套餐 5 小时能用 4 小时，现在半小时就快见底。首次订阅半价没了，只有 MAX 套餐才能用 GLM-5。这种"先让你爽，再慢慢收紧"的套路，我在闲鱼见多了。

去年我闲鱼爆单那天，一天 90 单，开心得不行。结果没过多久就被扣 20 分，封 7 天。平台的逻辑永远是：先让你尝到甜头，等你依赖了，再改规则。

好消息是 Kilo Code 提供了限免 GLM-5，TRAE CN 也火速上架了。这就是我喜欢这个行业的地方——总有人在找替代方案。我卖账号这么久，最大的感受就是：用户永远在找更便宜、更稳定的渠道，而我的工作就是帮他们找到。

## Agent 工具井喷，但我有点看不懂

今天日报里 Agent 相关的新闻特别多。Happycapy 让浏览器跑 Claude Code，GitCortex 一个总 Agent 调度多个 AI CLI。

说实话，这些我没太看懂。

我知道 Agent 是个大方向，大家都在说"2026 是 Agent 年"。但作为一个卖账号的，我更关心的是：这些工具火了之后，用户会需要什么账号？是 Claude 用得更多，还是 GPT？还是会出现新的需求？

我现在卖的主要是 Cursor Pro 换号器、Claude/Gemini/Codex 三合一、ChatGPT Plus 年卡这些。如果 Agent 工具真的爆发，我可能得调整一下产品线。但具体怎么调，我也不知道。

这就是我做这行的状态——永远在观察，永远在猜测，经常猜错，但也经常能抓住一些机会。

## 最后说点和日报无关的

最近大理天气不错，我每天下午会去洱海边走走。有时候想，我这个"卖铲子的人"能做多久？AI 发展这么快，说不定哪天账号这门生意就没了。

但转念一想，只要有信息差，就有生意做。DeepSeek 偷偷更新，有人知道有人不知道；GLM-5 调价，有人找到替代方案有人没找到。我的工作就是站在这些信息差的中间，帮用户省点事。

至于我那个"借 AI 实现长生不老"的疯狂梦想，今天的日报里没什么相关的。等 BioAI 日报吧，那个才是我真正关心的。
