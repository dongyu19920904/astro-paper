今天看日报的时候正在大理的咖啡馆里发呆，刷到 Anthropic 发布 Claude Code Security 的消息，脑子里第一个念头不是"哇好厉害"，而是"这东西要单独收费吗？我的 Claude 账号套餐要不要调整？"

卖了 130 多天账号，职业病已经刻进 DNA 了。

## Table of contents

## 安全工具这事，我其实看不太懂

说实话，Claude Code Security 具体怎么扫漏洞、怎么生成补丁，技术细节我没太看明白。但有一点我能感受到——传统安全扫描工具确实不太行。

去年我帮一个客户搭过一个小项目（用 Cursor 写的，代码质量你懂的），他后来拿去给安全团队审计，结果扫出来一堆"高危漏洞"，仔细一看全是误报。安全团队的人跟我说，这些工具就是"宁可错杀一千"，真正藏得深的问题反而找不到。

所以 Anthropic 说能找到"传统工具找不到的漏洞"，我是信的。AI 理解代码逻辑的能力确实比正则匹配强太多了。

但这东西目前还在限量测试，我估计短期内不会影响普通用户的账号价格。先观望。

## Agent 工具井喷，我有点焦虑

今天日报里最让我坐不住的，其实是那两个 Agent 框架——superpowers 56K Star，Composio 27K Star。

这说明什么？说明大家都在想办法让 AI "真正干活"，而不只是聊天。

我焦虑的点在于：如果 Agent 真的能自己调用各种 API、自己完成任务，那还需要人工买账号吗？

想了想，又觉得自己想多了。

就像当年 DeepSeek 刚出来的时候，我觉得"免费的这么强，谁还买 Claude"，结果呢？该买的还是买。因为大部分人要的不是"最强"，而是"能用"。Agent 工具再牛，普通用户也得先有个账号才能跑起来。

而且说实话，我自己试过几个 Agent 框架，配置起来挺折腾的。对于不想折腾的人来说，直接买个现成账号用 Claude 聊天，可能还是最省心的方案。

## Qwen 7.9 元撸 18000 次，这波我要不要进货？

日报里还有一条让我眼前一亮：Qwen Coding Plan 40 元套餐实付 7.9 元，能用 18000 次。

作为卖账号的，我的第一反应是：这价格，我能不能批量搞一波？

但转念一想，Qwen 在国内用户里的认知度还是不如 Claude 和 ChatGPT。我之前试过在闲鱼上架国产模型的账号，销量惨淡。大家还是认"洋品牌"。

不过这个价格确实香。我自己可能会买一个来用，写课程的时候当备用。1M 上下文，分析长文档应该很爽。

等等，我好像又在想怎么省钱了，而不是怎么赚钱。

## 语音 AI 卷成这样，我该不该入场？

今天日报横评了 5 款语音 AI 工具，看得我有点心动。

我一直想做有声书相关的副业，但之前觉得技术门槛太高。现在 GPT-SoVITS 5 秒就能克隆声音，Fish Speech 效果也很能打，感觉普通人也能玩了。

但我又想起去年写 AI 课程的惨痛经历——三个课程加起来几十万字，总共卖了不到 30 块。

内容创作这条路，我已经被教育过了。

还是老老实实卖账号吧。至少这个生意模式简单：进货、上架、发货、售后。不用绞尽脑汁想"用户想看什么内容"。

## 今天的碎碎念

在大理待久了，有时候会忘记外面的世界卷成什么样。

看完今天的日报，感觉 AI 行业又往前跑了一大步。Agent 工具在井喷，安全工具在进化，语音 AI 在内卷，连印度的 Codex 用户都两周涨了 4 倍。

而我呢，还在咖啡馆里想着"这条新闻会不会影响我卖账号"。

说起来有点好笑，但这就是我的真实状态。我不是什么 AI 专家，也不是技术大佬，就是一个在 AI 淘金热里卖铲子的普通人。

铲子卖得还行，月入稳定几千，够在大理活着。

这样也挺好的。
