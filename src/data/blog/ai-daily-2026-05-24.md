那个 GitHub 仓库一天涨了五万星，我盯着数字看了好几秒

卖 AI 账号这行，我最怕的不是竞争对手，是用户突然不需要我了。今天看到 claude-code-best-practice 单日涨了五万多 Star，第一反应不是"哇好厉害"，而是——完了，Claude 账号的需求是不是要变？然后我仔细想了想，发现自己又想多了，但这个思考过程本身挺有意思的，值得写出来。

## Table of contents

## 五万 Star 这个数字，我反复想了很久

[claude-code-best-practice](https://github.com/shanraisshan/claude-code-best-practice) 这个仓库，不是新模型，不是新工具，就是一份文档——教你怎么用 Claude 写代码。单日五万四千多 Star。

我在这行待了一年多，见过各种爆款。但一份"使用说明"能爆成这样，说实话有点出乎我意料。

然后我想起来，我当年在闲鱼卖提示词的时候，9.9 一份，一份要写半天。那时候我觉得提示词工程是个很有价值的东西，结果卖了没多久，市场就冷了——不是因为提示词不重要，而是大家开始觉得"这东西我自己也能写"。

现在这个仓库爆了，说明什么？说明大家又意识到"我自己写不好"了。

这个循环挺有意思的：工具越来越强 → 用户以为自己会用 → 发现用不好 → 疯狂找方法论 → 方法论又变成新的产品。

我当年卖提示词卖亏了，但方向没错，只是早了两年，而且定价策略完全错了。9.9 一份，卖的是"内容"，但用户真正愿意付钱的是"能用起来的系统"。这个道理我现在才算真正想明白。

## 美团那个虚拟人，我想的是另一件事

美团开源了 LongCat-Video-Avatar 1.5，音频驱动虚拟人，被说是目前开源里效果最好的。

<video controls preload="metadata" playsinline style="max-width:100%; height:auto;" src="https://video.twimg.com/amplify_video/2058141473472946176/vid/avc1/1024x384/C4R54zW3tFfXSNoZ.mp4?tag=27"></video>

技术细节我没仔细看，HuggingFace 上的模型我也没跑过，我不是这方面的专家。

但我想到的是：这东西开源之后，会有一批人拿去做直播数字人，然后会有一批人卖"数字人搭建服务"，然后会有一批人卖"数字人账号"或者"数字人工具账号"。

这条链路我太熟了。每次有新的开源模型出来，最先赚到钱的往往不是做技术的，而是做"让普通人用上这个技术"的中间层。

我现在卖的 Cursor、Claude、Windsurf 这些账号，本质上就是这个中间层。技术在那里，但普通人用不上——要么注册麻烦，要么支付麻烦，要么不知道怎么选。我解决的就是这个问题。

虚拟人这个方向，我暂时没打算进。不是因为不看好，而是我现在线程已经够多了，再加一条我真的转不过来。但我会盯着，等它变成一个"普通人也想用但用不上"的东西，那时候可能是个机会。

## Codex 看截图还原产品，这件事让我有点坐不住

有人把微信消息驾驶舱的截图丢给 Codex，让它直接复刻出来。

![Tweet Image](https://pbs.twimg.com/media/HJBVEfgaMAAVDI-.jpg)

我看到这个的时候，脑子里第一个念头是：那我的发卡站，是不是也可以这么搞？

我现在的发卡站是用现成系统搭的，有些地方用起来不顺手，但我不会写代码，改起来很麻烦，每次都要找人帮忙或者凑合用。如果 Codex 真的能"看截图还原产品"，那我是不是可以截一张我理想中的界面，让它帮我写出来？

我承认我对这个能力的边界完全不清楚。可能实际用起来没那么神，可能需要很多调试，可能我这个水平根本驾驭不了。但这个方向是对的——AI 编程工具正在把"有想法但不会写代码"这个门槛往下压。

我用 Cursor 写过一点东西，说实话写得很烂，但能跑。这已经是一年前的我完全做不到的事了。

## 今天状态一般，但有一件事想说

DeepSeek-V4-Pro 折扣永久化了。原来说到 5 月底截止，现在直接宣布永久打折。

这条新闻很多人可能觉得无聊，但对我来说有点感慨。

我第一次卖账号，就是 2025 年初 DeepSeek 爆火那会儿，在闲鱼搬运 API 赚差价，赚了几十块。那时候我完全不知道自己在做什么，就是看到有人买，我就去找货，然后加价卖。

现在 DeepSeek 的 API 价格一降再降，当年那种靠信息差赚差价的玩法早就不行了。但我也因为那次经历，慢慢摸索出了现在这套东西。

所以有时候我觉得，判断错了不一定是坏事。我当年觉得 DeepSeek 火不了，结果它火了，然后我因为它入了这行。如果我当时判断对了，可能就不会去试，也就没有后来的事了。

当然这不是说判断错了是好事，只是说——有时候被现实打脸，反而是个机会。

前提是你得在场。
