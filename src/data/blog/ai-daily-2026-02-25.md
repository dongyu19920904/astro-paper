今天的日报炸了。Anthropic 直接点名 DeepSeek、月之暗面、MiniMax 蒸馏 Claude，马斯克在评论区阴阳怪气。作为一个卖 AI 账号的，我看这条新闻的心情特别复杂——一边是我的供应商，一边是我的货源。

## Table of contents

## 蒸馏这事，我早就知道

说实话，"蒸馏"在圈子里根本不是秘密。

我卖账号这一年多，跟各种供应商打过交道，有些事情大家心照不宣。比如为什么某些国产模型突然就"变聪明"了？为什么有些 API 的输出风格跟 Claude 那么像？

但没人敢公开说。

今天 Anthropic 直接撕破脸了——2.4 万个账号，1600 万次对话。这数字看得我头皮发麻。我在闲鱼卖账号最疯狂的时候，一天也就 90 单，人家直接搞 2.4 万个账号批量薅。

马斯克那句评论太损了："他们怎么敢偷 Anthropic 从人类程序员那里偷来的东西？"

这话损归损，但确实戳到了一个尴尬的点——AI 公司用全网数据训练模型，现在又指责别人用自己的输出训练模型。这个逻辑链条，怎么说呢，有点套娃。

## 我的货源会不会出问题？

看到这条新闻，我第一反应是：我卖的 Claude 账号会不会受影响？

我在 aivora.cn 卖的 Claude/Gemini/Codex 三合一账号，货源一直挺稳定的。但 Anthropic 这次明显是要动真格了，如果他们加强账号审核，批量注册的难度肯定会上升。

去年我就被坑过一次。有个供应商一声不吭跑路了，那天我刚好卖了十几单，差点全退款。从那以后我就学乖了，货源至少要有两三个备选。

现在看来，这个策略是对的。

不过话说回来，Anthropic 点名的是"工业级蒸馏"，跟我们这种零售账号应该不是一回事。但谁知道呢，大公司一旦开始收紧，小虾米往往是第一批遭殃的。

## 月之暗面 20 天收入超去年全年

同一天还有另一条新闻：Kimi K2.5 发布 20 天，收入超过 2025 年全年。

这两条新闻放在一起看，味道就很复杂了。

一边是 Anthropic 指控月之暗面蒸馏，一边是月之暗面业绩爆发。杨植麟说"资金充裕，专注技术迭代"。我不知道这算不算一种回应，但确实很有底气。

作为一个卖账号的，我其实挺矛盾的。国产模型越强，我卖海外账号的需求可能就越少。但反过来想，AI 整体越火，用户越多，我的生意也不会差到哪去。

去年 DeepSeek 刚出来的时候，我还在闲鱼搬运 API 赚差价，赚了几十块。当时我觉得这玩意儿火不了，结果被打脸了。所以现在我学会了一件事：不要轻易预测 AI 行业的走向，因为我的判断经常是错的。

## GPT-5.3-Codex 开放了，我要不要进点新货？

今天还有一条：GPT-5.3-Codex 正式开放给所有开发者。

这个我得认真考虑一下。我现在卖的主要是 Cursor Pro 换号器和 Claude 账号，OpenAI 的产品线一直没怎么碰。原因很简单——ChatGPT Plus 的供应商太多了，价格已经卷到没什么利润。

但 Codex 不一样。这是专门给开发者用的，用户群体更精准，愿意付费的比例也更高。

问题是我自己对 Codex 不太熟。我用 Cursor 写代码（虽然很菜），用 Gemini 写过课程，但 Codex 的 API 我还没怎么玩过。

可能得先自己试试，再决定要不要进货。卖自己不懂的东西，迟早要翻车。

## 今天的感受

写到这里，发现今天的日报其实就一个主题：AI 公司之间的战争升级了。

Anthropic 撕破脸、韩国电视台起诉 OpenAI、版权争议越来越多。这些大公司打架，我们这些小卖家只能看着。

但有一点我是确定的：越乱，机会越多。

去年闲鱼封号那次，我被扣了 20 分，封了 7 天。当时觉得天都塌了，结果逼着我做了自己的网站 aivora.cn，现在反而更稳定了。

所以今天看到这些新闻，我没有特别焦虑。该卖的账号继续卖，该备的货源继续备。大公司打架，我就在旁边卖铲子。
