今天的日报炸出了一个让我特别有感觉的数字：Claude Code 月费200 美元，实际算力成本 5000 美元，Anthropic 每收一个用户倒贴 4800 美元。作为一个卖了 130 多天AI 账号的人，我看到这个数字的第一反应不是震惊，是似曾相识。

## Table of contents

## 补贴大战，我太熟了

说真的，看到 Cursor 进入"战时状态"那条新闻，我在大理的咖啡馆里差点把奶茶喷出来。

不是觉得好笑，是觉得——这剧本我见过啊。

2025 年初DeepSeek 爆火的时候，我在闲鱼搬运 API赚差价。那时候各家模型的 API 价格战打得昏天暗地，今天你降价明天我免费，我夹在中间赚的那几十块钱就是在别人的补贴里捡漏。现在 Anthropic 用每个月倒贴 4800 美元的方式抢用户，本质上和当年外卖大战、打车大战一模一样。

但有个关键区别：外卖补贴一单亏几块钱，AI 算力补贴一个用户亏几千美元。这烧钱速度，哪怕是 Anthropic 这种融了几十亿美金的公司，也不可能永远扛下去。

所以我的判断是——**现在是囤 Claude Code 账号的好时机**。趁补贴还在，能用就用。等哪天 Anthropic 扛不住了开始涨价，或者砍功能，到时候再说。

当然了，我之前判断 DeepSeek 火不了，结果被打脸打得生活不能自理，所以这个判断大家随便听听就好。

## 我观察到的一个有意思的现象

做了这么久账号生意，有个变化很明显：**用户越来越不忠诚了**。

以前一个人买了 Cursor Pro，基本上就一直用Cursor。现在呢？同一个客户，这个月买 Cursor，下个月问我有没有 Claude Code，再过两周又来问 Windsurf 怎么样。

[Forbes 那篇报道](https://x.com/dotey/status/2030363129096397123)说Cursor 的企业客户撑起了 60% 的收入，企业切换工具流程漫长，给了 Cursor 喘息窗口。这我信。但个人用户这边，切换成本几乎为零。今天 Claude Code 好用，明天就跑了。

这对我来说其实是好事。用户跳来跳去，每跳一次就得找我买一次账号。**工具越卷，我生意越好。** 听起来有点缺德，但卖铲子的人就是这样——矿工打架打得越凶，铲子卖得越好。

## /loop 命令让我想起一件事

Claude Code 上线了[/loop 命令](https://x.com/dotey/status/2030199144388722949)，说白了就是让AI帮你值班，最长盯三天。

这让我想起我自己的一个痛点。我在aivora.cn 卖账号，最头疼的事情之一就是——**半夜有人下单，发货不及时，客户就急了**。我试过各种自动化方案，但总有些边角情况需要人工处理。

如果以后真的能用 AI 帮我 7×24 盯着订单系统，自动处理常见问题，那我的生活质量能上一个台阶。

不过看了[反编译分析](https://x.com/dotey/status/2030360433228415460)之后，发现 /loop 底层就是个 cron 包装器，关掉终端就没了。这……离我想要的"AI 客服"还差十万八千里。

但方向是对的。我相信再过半年，这种东西会好用很多。

## 今天最打脸的一条

[OpenClaw 作者说GUI 比 CLI 更香](https://x.com/dotey/status/2030391079780370507)，这条我必须单独说一下。

技术圈有个特别烦人的毛病：**什么新就吹什么，然后急着宣判旧东西的死刑**。前段时间到处都在喊"GUI 已死，未来是 Agent + CLI的天下"，结果做CLI 工具的人自己都觉得 GUI 更好用。

我作为一个非技术背景的人，说实话一直觉得 CLI 那种黑乎乎的界面挺劝退的。但每次这么说，都会被"懂技术的人"嘲笑。现在好了，人家自己打自己脸了。

## 随便说两句

今天大理下雨了，在咖啡馆写这篇的时候窗外雾蒙蒙的。

最近生意还行，三月份开局不错。QQ 适配 AI Agent 那条新闻我也注意到了——如果真的能在 QQ 里直接控制 Claude，那国内用户的门槛又低了一截，对我来说可能是个新的增长点。

不过说实话，这个行业变化太快了。我 130 天前开始卖账号的时候，Cursor 还是绝对霸主，现在就已经进入"战时状态"了。再过 130 天会怎样？我不知道。

我唯一确定的是：**不管谁赢，工具总得有人用，账号总得有人卖。**

铲子生意，永不过时。卖铲子的人看AI 编程大战：Anthropic 每收一个用户倒贴 4800 美元，我笑了

今天的日报炸出一个让我心跳加速的数字：Claude Code 月费 200 美元，实际算力成本 5000 美元，Anthropic 每收一个用户倒贴 4800 美元。作为一个卖 AI 账号的人，我第一反应不是"哇好厉害"，而是——这钱烧完之后，我的供应商还活着吗？

## 这场补贴战，我太熟悉了

看到 Cursor 进入"战时状态"的新闻，我愣了一下，然后笑了。

不是幸灾乐祸。是因为这个剧本我在自己的小生意里演过——虽然规模差了十万八千里。

去年底有一阵子，闲鱼上突然冒出来一堆卖 Claude账号的，价格从我的六折往下砍。我当时慌了，跟着降价，结果发现那些人根本不是在做生意，他们是在清库存，或者纯粹是新手不懂定价，亏着卖赚个好评。

我扛了两周，那帮人消失了。价格回来了。

所以当我看到 [Anthropic 每个用户倒贴 4800 美元](https://x.com/dotey/status/2030182421287592420)的时候，我的直觉反应是：这是互联网最经典的打法，先烧死对手再涨价。但问题是，AI 的算力成本跟打车外卖不一样。你补贴一个用户一个月亏 4800 美元，十万个用户就是每月4.8 亿美元。

这个数字我算了三遍，还是觉得离谱。

当然，Anthropic 背后站着亚马逊，不差这点钱。但作为一个卖铲子的人，我真正担心的不是谁赢——我担心的是打仗期间，子弹乱飞，我的供应链被波及。

去年就出过一次。某个供应商突然跑路，一声不吭，我那天刚好卖了十几单，差点全退款。后来花了三天才找到替代的。从那以后我学乖了，同一个产品至少备两个供应商。

## Cursor 的焦虑，其实也是我的焦虑

[Forbes 那篇报道](https://x.com/dotey/status/2030363129096397123)说Cursor 个人订阅在亏钱，企业客户撑起了 60% 的收入。

这句话让我想起我自己的处境。我在 [aivora.cn](https://aivora.cn) 上卖的产品，Cursor Pro 换号器一直是大头。如果 Cursor 真的被Claude Code 打趴了，我这条产品线就危险了。

但转念一想——又没那么简单。

我观察到一个有意思的现象：来我这里买账号的用户，大部分人其实搞不清楚 Cursor、Claude Code、Windsurf 这些工具到底有什么区别。他们的需求很朴素：**我要一个能帮我写代码的东西，你给我推荐一个就行。**

所以这场大战打到最后，对我这种卖铲子的人来说，可能影响没那么大。A 工具倒了，用户流向 B 工具，我就卖 B 工具的账号。铲子还是铲子，只是换了个牌子。

当然这是乐观的说法。悲观的说法是——如果 Claude Code 把价格压到人人都能直接官方订阅，谁还需要我这个中间商？

我也不知道。走一步看一步吧。

## /loop 这个功能让我有点心动

说回技术本身。[Claude Code 的 /loop 命令](https://x.com/dotey/status/2030199144388722949)我认真看了一下，虽然技术细节没完全看懂，但大概意思我理解了——你可以让 AI帮你盯着某个任务，出了问题自动修。

这个我是真需要。

我的小店后台有些自动化脚本，偶尔会挂。现在的处理方式是我手机上设了个监控，报警了就去看。但经常是大理凌晨三点报警，我第二天早上才处理，客户等了一夜。

如果 AI 能帮我值班，哪怕只是"发现脚本挂了→自动重启→给我发条消息"，那就值回票价了。

不过[有人扒了底层](https://x.com/dotey/status/2030360433228415460)，说其实就是个 cron 包装器，关掉终端就没了。那就……先观望吧。

## 一个跑题的想法

今天还看到 [OpenClaw 作者说GUI 比 CLI 更香](https://x.com/dotey/status/2030391059780370507)。

![GUI vs CLI](https://pbs.twimg.com/media/HC0uxcLWUAAlY-4?format=jpg&name=orig)

这条让我笑出声。因为我之前在闲鱼上，遇到过一个客户跟我吵架，说"真正的程序员都用命令行，你推荐带界面的工具是看不起我"。结果后来他自己私信我说还是想要带界面的版本。

技术圈这种"鄙视链文化"真的挺搞笑的。好用就是好用，管它是 GUI 还是 CLI。我一个卖账号的都看得明白的道理，不知道为什么总有人要争。

说到底，今天日报最让我在意的就一件事：**这场烧钱大战能持续多久？** 作为一个在夹缝中卖铲子的人，我既希望它持续久一点（补贴期用户多），又怕它结束得太突然（供应链断裂）。

大理今天下雨，咖啡馆人不多。写完这篇，该去处理几单发货了。
