今天看日报，Gemini 大规模封号的消息让我心里一紧。不是因为我自己被封了，而是想起上个月我刚进了一批 Gemini 号，还没来得及出手呢。

## Table of contents

## 卖账号的人最怕什么

说实话，卖 AI 账号这行，最怕的不是没客户，是供应商那边出问题。

日报里说"10 个号 3 个暴毙"，这个比例其实还算温和。我去年经历过更惨的——有一次 Claude 那边风控升级，我手里二十多个号一夜之间全红了。那天我刚好卖了十几单，差点全退款。

后来我学乖了，不囤太多货。但问题是，不囤货就意味着供应商涨价你只能认，断货你只能干等。这行就是这样，两头受气。

看到有人说"保住手里有的就是胜利，别再折腾了"，我深有同感。薅羊毛这事，红利期过了就是过了。我当年在闲鱼搬运 DeepSeek API 赚差价，也就火了那么几天，后来大家都知道怎么弄了，利润直接归零。

## 英伟达和 OpenAI 的事让我想到一个问题

千亿美元的交易黄了，黄仁勋私下吐槽 OpenAI"缺乏商业纪律"。

这句话我反复看了几遍。

我一个卖账号的，当然不懂什么叫"商业纪律"。但我知道一件事：Altman 这种"先造势再谈判"的风格，在我们这行叫"画饼"。

我见过太多这样的人了。说要搞大项目，说要长期合作，结果聊到付款环节就消失了。或者更常见的——先把声势造起来，让你觉得不合作就亏了，然后再慢慢谈条件。

不是说这招不好使，但用多了，信用就没了。

英伟达转头给 Anthropic 投了 100 亿，这个信号挺明显的。我自己卖的账号里，Claude 系列的需求确实在涨。不知道是不是因为 Claude Code 最近太火了。

## Claude Code 配置这事，我也被折腾过

日报里说 Clawdbot 配置"堪称噩梦"，用户集体崩溃。

我太懂这种感觉了。

我不是开发者，但我卖 Cursor Pro 换号器，经常要帮客户解决各种奇怪的问题。有时候明明按教程来的，就是不行。后来发现是配置文件里多了个空格，或者 URL 少了个斜杠。

这种问题最烦人的地方在于：你不知道问题出在哪，只能一个一个排查。排查半天发现是个特别蠢的错误，又气又好笑。

所以我现在特别理解为什么 CC-Switch 这种工具能火。能帮用户省掉这些折腾的东西，就是好东西。

说到这个，我在想要不要进点 Claude Code 相关的产品。但说实话，我还没搞清楚这玩意儿的商业模式。它是订阅制的吗？能不能像 Cursor 那样做换号器？我得再研究研究。

## 今天没什么特别深刻的想法

写到这里发现，今天的日报虽然信息量很大，但没有哪条让我特别兴奋或者特别焦虑。

可能是因为最近生意稳定了，心态也稳定了。

在大理待了几个月，慢慢习惯了这种节奏。早上起来看看日报，处理几单生意，下午去咖啡馆发呆，晚上写写东西。月入几千，不多，但够用。

有时候会想，这样下去行不行？AI 这行变化太快了，今天卖得好的东西，明天可能就没人要了。

但想太多也没用。先把手里的事做好吧。

对了，日报里提到元宝发红包，我去领了一下，还真有。虽然不多，但白嫖的快乐是真实的。
