今天的日报让我有点兴奋，不是因为技术突破，而是因为神仙打架。Sam Altman 凌晨发长文怼 Anthropic，这种级别的互撕在 AI 圈还是头一回见。作为一个卖账号的，我第一反应居然是——我的 Claude 账号要不要涨价？

## Table of contents

## 神仙打架，卖铲子的最开心

说实话，看到 Sam Altman 那条推的时候，我正在大理的咖啡馆里发呆。

"服务富人的昂贵产品"——这句话太狠了。Anthropic 花大价钱投超级碗广告，结果被 OpenAI 老板亲自下场回怼，说你们才是"想控制 AI 规则的独裁公司"。

我不站队，因为我两边的账号都卖。

但我得承认，这种级别的互撕对我这种卖铲子的人来说是好事。为什么？因为用户会更关注这两家的产品，会想亲自试试到底谁更强。试用需要账号，账号从哪来？嘿嘿。

去年我刚开始卖 Claude 账号的时候，很多人问我："Claude 和 ChatGPT 有什么区别？"我说不清楚，因为我自己也搞不太明白。但现在不一样了，两家公司自己下场打架，帮我做了最好的市场教育。

## Codex 一周 50 万下载，我有点慌

这个数据让我有点焦虑。

我现在卖的主力产品是 Cursor Pro 换号器，但 OpenAI 的 Codex 势头太猛了。50 万下载，一周。Sam Altman 还说"接下来几周会让开发者爱上"。

我在想一个问题：如果 Codex 真的比 Cursor 好用，我是不是该提前进点 Codex 相关的货？

但问题是，Codex 现在还没有像 Cursor 那样的"换号器"需求。Cursor 的商业模式是订阅制，用户想省钱就需要换号；Codex 目前看起来是走 API 路线，商业模式不一样。

我也不知道该怎么判断。去年我觉得 DeepSeek 火不了，结果被打脸。这次我不敢轻易下结论了。

先观望吧，反正 Cursor 的需求还在，我的换号器还能卖。

## Sonnet 5 今晚发？我得盯着供应商

V0 团队暗示 Claude Sonnet 5 今晚发布，这个消息让我立刻给供应商发了条消息："最近 Claude 账号稳定吗？"

为什么？因为每次大版本更新，都可能出幺蛾子。

我之前有个供应商，Claude 3.5 发布那天突然跑路了。那天我刚好卖了十几单，差点全退款。后来才知道，他的账号来源出了问题，新版本上线后批量被封。

所以现在我学乖了，大版本更新前先问一嘴。虽然供应商说"没问题"，但我还是会多备点库存，以防万一。

卖账号这行，最怕的不是没客户，是供应链出问题。客户可以慢慢来，但账号断供了，口碑就没了。

## 一个让我有点感慨的细节

今天日报里有条新闻：Clawdbot 新版支持飞书连接了。

这个功能对国内用户来说太实用了。以前想用这类 AI Agent，得开 Discord 或 Telegram，很麻烦。现在能在飞书里用，门槛一下子降低了。

我感慨的是什么呢？

一年前我刚入行的时候，在闲鱼卖 AI 指令，9.9 一份，一份要写半天。那时候我觉得"AI 工具"就是 ChatGPT 加几个提示词。

现在呢？AI Agent、MCP、Hooks、Skills 目录……我已经看不懂一半的技术名词了。

但神奇的是，我的生意反而越来越好。因为工具越复杂，普通用户越需要"开箱即用"的解决方案。他们不想研究什么 Hooks 机制，他们只想要一个能用的账号。

所以我现在的定位很清晰：我不是技术专家，我是"最后一公里"的服务商。技术的事让大厂去卷，我负责把成品送到用户手里。

## 今天的随想

大理最近降温了，咖啡馆里的人比前几天少了一些。

写这篇博客的时候，我一直在想：AI 圈的神仙打架，对我这种小卖家来说意味着什么？

答案可能是：什么都不意味着，也什么都意味着。

不意味着什么，是因为无论谁赢，用户都需要账号，我的生意模式不会变。

什么都意味着，是因为这个行业变化太快了，今天的爆款可能明天就过时。我得一直盯着，一直调整。

好累，但也挺刺激的。

明天继续卖账号，继续看神仙打架。
