Brex 的裁员让我松了口气，看来卖铲子还能再干两年

在大理呆久了，人容易变懒。今天本来打算去苍山脚下发呆，结果刷到两条新闻，直接让我从躺椅上坐了起来。

一条是看乐子的，一条是看门道的。

先说乐子，OpenAI 前 CTO Mira Murati 的新公司，全员会上因为炒人，结果两位联创当场在 Slack 上辞职。这剧情比电视剧还狗血。说实话，这让我想起我做“号贩子”这行遇到的那些上游供应商。

前一天还在群里吹牛逼说“独家渠道，稳如老狗”，第二天可能就注销账号跑路了。**在这个行业里，没有任何关系是稳固的，不管是硅谷的顶流创业公司，还是我那些躲在阴暗角落里的 API 供货商。** 所以我现在备货从来不敢只压一家，Cursor 的号我都备了三个渠道，毕竟用户不管你上游死没死，他号登不上就是要骂我。

但真正触动我的是另一条新闻：**Brex 靠 AI 提效，年收入干到了 5 亿美元，但工程团队人数没变。**

这就很有意思了。

## 不招人，只买号？

新闻里说，Brex 的策略是“不堆人头，堆 AI”。他们不强制员工用什么工具，ChatGPT、Claude、Gemini 随便选，Cursor、Windsurf、Claude Code 也随便用。

看到这儿我笑了。**这不就是我那帮客户的真实写照吗？**

我后台数据显示，最近买 Cursor Pro 换号器和 Claude 独享号的人，很多都是大厂或者独角兽公司的程序员。为什么？因为公司报销流程太慢，或者干脆只提供统一的（通常是不好用的）官方工具。

程序员是有洁癖的。既然 Brex 这种公司都证明了“让员工自己选工具”能把效率提升 30%-100%，那说明**工具的碎片化**是必然趋势。

以前我觉得自己是在赚信息差的钱，现在我觉得自己是在给这些想提效的个体“递铲子”。只要大公司还在纠结数据安全不敢全员买 Cursor，只要正版订阅费对个人开发者来说还有点肉疼，我这个生意就能一直做下去。

Brex 的成功，某种程度上验证了我的商业模式：**未来的公司不会为所有人买所有的单，但所有人都会为了自己的效率去买最好的单。**

## 9.9 元提示词的坟墓

还有条新闻是关于 FLUX.2 的，说它生成图快，而且交互简单，不需要复杂的提示词。

看到这儿，我心里其实挺复杂的。

大家可能不知道，我两年前是卖提示词的。当时在闲鱼，一份《Midjourney 咒语大全》卖 9.9，我还觉得自己是“提示词工程师”，特别牛逼。那时候每天就在研究怎么把 prompt 写得像代码一样复杂。

现在呢？FLUX.2 这种工具告诉你：**别扯淡了，用户不需要学咒语，点两下就行。**

Brex 的新闻里也提到了，运营团队用 Retool 搭个 Agent 就能干活，根本不需要懂什么 Prompt Engineering 的深奥技巧。

我当年的那个“提示词工程师”的梦，算是彻底碎了。**在这个技术迭代速度下，任何依附于“工具缺陷”的技能都是短命的。** 以前工具笨，需要人会写 Prompt，现在工具聪明了，这技能就废了。

这也是为什么我现在只卖账号，不卖教程。**账号是通向算力的门票，而教程往往是过期的旧报纸。**
