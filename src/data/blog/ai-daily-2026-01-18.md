500 亿的公司说没就没，手里握着账号的我反倒踏实了

大理今天的阳光不错，我在洱海边的"床单厂"集市附近找了个咖啡馆蹲着。

本来想把昨天那个 Cursor Pro 换号器的 bug 修一下，结果这几天生意太好，客服消息回不过来，索性就摸鱼看了一眼今天的日报。

看完最大的感受是：**这也太魔幻了。** 硅谷那帮人搞宫斗比我们卖号圈子还乱，反倒是一条关于"工具自由"的新闻，让我看到了今年赚钱的方向。

## 硅谷版"提桶跑路"

Mira Murati 的那家 Thinking Machines Lab，估值冲着 500 亿美金去的，结果全员会上 CTO 被炒，两个联创直接在 Slack 上发帖辞职。

![Thinking Machines Lab 上演全员会议惊魂](https://pbs.twimg.com/media/G-58WdpW8AAG3yU?format=jpg&name=orig)

看着这张图，我没觉得多震惊，反倒有种莫名的亲切感。

为什么？因为这种"上一秒还在画饼，下一秒人去楼空"的戏码，我在 AI 账号的"灰产"圈子里见得太多了。

记得去年这个时候，我有个特别稳定的 Claude 逆向 API 供应商。那天早上我还给他转了 2000 块预付款，下午我想补货，发现 TG 群没了，头像灰了，消息发不出去了。那种感觉，估计和今天 Mira Murati 站在讲台上看到 Slack 弹窗时的感觉差不多——懵逼，然后想骂娘。

那个供应商手里也是握着所谓的"独家技术"，就像这些硅谷大佬握着模型权重一样。但在这个行业，**技术壁垒有时候脆得像张纸，真正的人心壁垒才是也是纸。**

这也更加坚定了我现在的策略：**我不依附于某一家具体的公司。** OpenAI 乱了我就推 Claude，Claude 封号我就推 Gemini。只要铲子还在卖，矿场老板换谁做其实无所谓。

为了长生不老，这种焦虑我是不吃的。少生闲气，多晒太阳，活得久才是硬道理。

## Brex 的"工具自由"是我的财富密码

相比于职场八卦，Brex 那条新闻才是我真正关心的。

这家公司靠 AI 把效率提了一倍，年收入干到 5 亿美金。重点是他们的策略：**不强制员工用什么工具，ChatGPT、Claude、Cursor、Windsurf 随便选。**

这就是我最近生意好的原因啊！

以前大家只买一个 ChatGPT Plus 就够了。现在呢？
- 写代码的，不仅要 Cursor，还要试一下 Windsurf，最近又有人问我 Kiro 怎么买。
- 写文档的，觉得 Claude 3.5 Sonnet 比 GPT-4o 好用。
- 搞长文本的，又得去薅 Gemini 1.5 Pro 的羊毛。

**"工具碎片化"就是"卖铲人"的红利期。**

前两天有个客户，是个刚毕业的程序员，在我这买了个 Cursor Pro 的独享号，又买了个 Claude 的拼车号。他问我："yuyu 哥，我是不是买重了？"

我说："兄弟，这不是买重了，这是现在的常态。就像 Brex 这种大厂都不限制工具了，你自己干活更得把武器库配齐。"

其实我心里没说的是：**我也分不清哪个更好。** 我自己写代码（虽然主要是用来修我的自动发货机器人），有时候觉得 Cursor 的 Composer 很爽，有时候又觉得 Windsurf 的上下文理解更准。

既然连我都分不清，那用户更分不清。分不清的结果就是——**成年人不做选择，我全都要。**

所以，2026 年我的判断是：**聚合类的账号（比如三合一）或者专注于垂直领域的"全家桶"会更好卖。** 用户不需要你告诉他哪个最好，用户需要的是极低的成本把所有工具都试一遍。

## 真正的"数字游民"不需要协作桌面

还有条新闻说 Eigent 搞了个开源协作桌面，把所有工作流统一到一个界面，适合数字游民。

说实话，作为在大理旅居了一年多的"资深游民"，我对这种工具持怀疑态度。

我们在大理这帮人，看起来是在海边敲代码、做设计，其实大部分时间是在发呆、喝茶、聊天。**我们不需要一个"超级大脑"把 Slack 和 Notion 整合在一起逼着我们高效工作。**

如果我想高效，我就回深圳坐班了。

我现在用的工具流极其简单：
1. **微信/闲鱼**：接单、挨骂、处理售后。
2. **Cursor**：写点脚本，维护网站。
3. **备忘录**：记录我的"长生不老"计划（最近在研究白藜芦醇到底是不是智商税）。

越复杂的工具，维护成本越高。Eigent 这种东西，听起来很美，但如果它崩了，或者它依赖的 API 变了，我就得花一天时间去修它。

对于想搞副业或者旅居的人，我的建议是：**别迷信工具。** 哪怕你用最原始的 Excel 表格，只要能把账记清楚，把客户服务好，赚的钱都比整天折腾"All-in-one 工作流"的人多。

Brex 这种大公司需要流程，个体户只需要**现金流**。
