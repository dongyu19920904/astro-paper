硅谷在演宫斗剧，我在大理卖铲子

今天大理的天气好得不像话，苍山上的雪还没化，我坐在露台上喝咖啡，顺手翻了翻今天的 AI 日报。

说实话，看完第一条新闻，我差点把嘴里的咖啡喷出来。

## 人类真的是草台班子

先说说那个让我想笑又觉得魔幻的新闻：[Mira Murati 的创业公司 Thinking Machines Lab 炸了](https://x.com/dotey/status/2012751148835389651)。

这姐们儿之前在 OpenAI 可是风云人物，出来创业估值奔着 500 亿去。结果呢？全员会上宣布开除 CTO，两个联创当场在 Slack 上辞职，接着又跑了俩，这帮人转头就回了 OpenAI。

你看，这就是我不愿意去大厂卷，宁愿在闲鱼和私域卖账号的原因。人类的组织永远充满了情绪、博弈和这种戏剧性的"宫斗"。技术再牛逼，只要是人在搞，就逃不开这些破事。

我想起去年在闲鱼上遇到的一个客户。他也是搞 AI 创业的，买了几个 Claude 的号说是给团队做测试。结果没过两周，他跑来跟我吐槽，说合伙人因为觉得 Claude 写的代码风格不如 GPT-4，两人大吵一架散伙了。

当时我就想：**AI 都在进化成神了，人类还在因为这种屁事内耗。**

这也是为什么我对"AI 生命科学"那么痴迷。我的那个人类长生不老的梦想，某种程度上也是因为对脆弱肉体和情绪化大脑的不信任。如果 AI 能接管我们的健康管理，甚至接管部分决策，是不是就能少点这种"全员会当场辞职"的闹剧？

## 真正赚钱的人在闷声发大财

比起硅谷的狗血剧，[Brex 的这条新闻](https://x.com/vista8/status/2012582126999179462)才是我作为"卖铲人"真正关心的。

这家公司年收入破 5 亿美金，但是工程团队人数居然没变。怎么做到的？**不招人，招 AI。** 他们不强制员工用啥工具，Cursor、Windsurf、Claude Code 随便用。

看到这儿我眼睛亮了。这就是我的生路啊！

过去这 130 多天，我在 aivora.cn 卖得最好的就是 Cursor Pro 的换号器和 Claude 的合租号。之前我还有点焦虑，担心 AI 会不会发展到一定程度，大家就不需要这么多工具了？

Brex 给了我一颗定心丸：**未来的公司，员工数量可能不会涨，但"AI 账号"的消耗量绝对会暴涨。**

以前一个程序员就是一个程序员。现在的程序员，是一个人 + 一个 Cursor 账号 + 一个 Claude 订阅 + 一个 Gemini 做备用。对于老板来说，买几个号才多少钱？比招一个人便宜太多了。

我后台的数据也印证了这一点。最近来买 Cursor 换号器的，很多不是个人开发者，而是小团队的负责人，一买就是五六个 key。他们算得很清楚：花几百块钱买工具，能让这几个人干出十个人的活，这买卖太划算了。

所以说，只要技术还在进步，只要老板们还想省钱，我这个"卖铲子"的生意就能一直做下去。

## 焦虑之后的平静

说回我自己。其实昨天我有点小郁闷，因为 DeepSeek 的 API 接口又波动了，导致我那边的几个自动化脚本挂了半天，赔了客户不少笑脸。

但看到今天 [Eigent 开源协作桌面](https://github.com/eigent-ai/eigent) 这种东西出来，我又觉得挺有意思。虽然我现在还在用传统的浏览器和本地工具，但这种"把所有工作流聚合到一个桌面"的想法，特别适合我们这种数字游民。

我现在在大理，每天的工作就是：
1. 起床看新闻（就像现在）
2. 检查账号库存（Cursor 没死吧？Claude 没封吧？）
3. 回复客户（"亲，这个报错是官方问题，换个节点试试"）
4. 研究新工具（为了以后能卖它）

如果有工具能把我这一套流程自动化了，我离我的"半躺平"目标就更近一步了。

有时候觉得，自己就像是个在淘金河边摆摊的小贩。看着河里的人（像 Mira Murati 她们）因为分金子打得头破血流，或者是像 Brex 那样挖到了大金块。

我呢？我不下水。我就坐在岸边，卖我的铲子和筛子。今天卖出去一个 Cursor 账号，赚的钱够我在大理吃顿好的菌子火锅，这就够了。

至于长生不老？那是 5 年后的 yuyu 要操心的事。现在的 yuyu，得去给那个刚下单的客户发货了。
