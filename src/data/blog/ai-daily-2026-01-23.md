今天刷日报的时候，看到智谱 GLM-4.7 跑通了 YouTube 视频自动剪辑 Skills，下载、翻译、烧字幕一条龙，自媒体人的偷懒神器开源了。第一反应不是"哇好厉害"，而是——卧槽，我 aivora.cn 上卖的那些 Claude 账号，是不是又要多一个卖点了？但紧接着又看到 CodePlan 的数据：国产模型 Tokens 用量已经超过 Claude 系列了。这让我有点慌，也有点兴奋。

## Table of contents

## Skills 生态起飞，我的账号要涨价了？

说实话，Claude Code 的 Skills 功能我自己也在用，但一直没太当回事。毕竟我不是专业开发者，写代码对我来说就是"能跑就行"的水平。但今天看到这个 YouTube 视频剪辑的 Skills，我突然意识到——这玩意儿的门槛正在疯狂降低。

以前做视频剪辑，要么花钱请人，要么自己学 Premiere，要么用那些半自动化的工具慢慢磨。现在呢？一个 Skills，自动下载视频、提取字幕、智能分段、翻译双语字幕、烧录进视频，还能生成社交媒体发布文案。最离谱的是，用智谱 GLM-4.7 测试，一次搞定，几乎零成本。

这对我来说意味着什么？**我卖的 Claude 账号，可能要多一个新的用户群体了——自媒体人。**

之前买我账号的，主要是程序员、写作者、学生党。但自媒体人这个群体，需求量可能更大。毕竟视频剪辑是刚需，而且频率高。一个做 YouTube 搬运的，一天可能要剪好几条视频。如果 Skills 真的能做到"一键搞定"，那这个需求会爆炸。

但问题来了——**Skills 生态起飞，对我来说是好事还是坏事？**

好的一面是，用户需求增加，我的账号销量可能会涨。坏的一面是，竞争也会更激烈。毕竟 Skills 是开源的，任何人都能用。而且，今晚 8 点还有大佬直播教学，门槛进一步降低。这意味着，可能会有更多人进入这个市场，卖账号的竞争会更卷。

我现在的策略是：**先观望，再调整。** 如果 Skills 真的火了，我可能会在 aivora.cn 上单独开一个"Skills 专用账号"的分类，主打自媒体人群体。但如果只是昙花一现，那我就继续卖我的 Cursor Pro 和 Claude 三合一账号，稳扎稳打。

## 国产模型崛起，我的供应链要调整了

今天看到的另一条新闻，让我更慌——**CodePlan 的用户已经开始用国产模型替代 Claude 了。**

过去半个月，GPT/Gemini + 国产开源模型的 Tokens 用量已经远超 Claude 系列，并且每日消耗还在上升。这说明什么？**国产模型在编程场景的实战能力已经能打了。**

这对我来说是个信号：**用户的选择正在变化。**

我之前一直觉得，Claude 是编程场景的王者，用户离不开它。但现在看来，Claude 的供应链不稳定（之前断供过好几次），加上国产模型的崛起，用户正在寻找替代方案。而且，国产模型的价格更便宜，对于高频使用的用户来说，成本优势明显。

这让我开始思考：**我是不是该调整一下我的产品线？**

目前我在 aivora.cn 上卖的主要是 Claude、ChatGPT、Gemini 这些国外模型的账号。但如果国产模型真的起来了，我是不是也该进点国产模型的账号？比如智谱 GLM、Minimax、Kimi 这些。

但问题是，**国产模型的账号好卖吗？**

我之前试过卖 DeepSeek 的 API，赚了几十块，但销量一般。因为国产模型的用户群体，很多都是技术玩家，他们更倾向于自己搞定，而不是买账号。而且，国产模型的价格本来就便宜，我能赚的差价很少。

所以，我现在的想法是：**国产模型可以作为补充，但不能作为主力。** 我的主力还是 Claude、ChatGPT、Gemini 这些国外模型，因为它们的用户群体更广，付费意愿更强。但我会关注国产模型的动态，如果哪天它们真的成为主流，我会立刻调整策略。

## 今晚的直播，我要不要去听？

今天看到歸藏和 Listenhub 今晚 8 点有 Skills 教学直播，我在想：**我要不要去听？**

说实话，我对 Skills 的技术细节不太懂。我不是专业开发者，写代码对我来说就是"能跑就行"。但作为一个卖账号的，我需要了解用户的需求。如果 Skills 真的火了，我需要知道用户会怎么用，他们会遇到什么问题，我能提供什么帮助。

所以，**我决定去听。** 不是为了学技术，而是为了了解市场。

我想知道：
- Skills 的门槛到底有多低？普通用户能不能上手？
- 哪些场景最适合用 Skills？自媒体人会不会真的买账？
- Skills 对账号的消耗有多大？用户会不会因为 Skills 而增加购买频率？

这些问题，可能会直接影响我的生意。所以，今晚 8 点，我会准时蹲守直播间。

## 我的小白困惑：AI 工具越来越多，我该怎么选？

最后说个题外话。今天看到 AionUi 这个开源工具，支持 Gemini CLI、Claude Code、Codex 等多种 AI 工具。我第一反应是：**AI 工具太多了，我自己都分不清哪个好。**

我现在用的主要是 Cursor Pro 和 Claude。Cursor 用来写代码，Claude 用来写文案和做一些简单的数据分析。但每次看到新工具，我都会想：**这个是不是比我现在用的更好？我要不要换？**

但问题是，**我没时间一个个试。** 我每天要处理订单、回复客户、维护供应链，还要写博客、刷日报。哪有时间去折腾新工具？

所以，我现在的策略是：**够用就行，不追新。** 除非某个工具真的能解决我的痛点，否则我不会轻易换。

但这也让我意识到，**很多用户可能和我一样，不知道该选哪个工具。** 这对我来说，可能是个机会。如果我能在 aivora.cn 上提供一些"工具选择指南"，帮用户找到最适合他们的工具，可能会增加用户粘性。

这个想法还不成熟，但我觉得值得试试。

## 最后的碎碎念

今天的日报让我有点焦虑，也有点兴奋。焦虑的是，AI 工具的变化太快了，我不知道我的生意能不能跟上。兴奋的是，变化意味着机会，只要我能抓住，可能会有新的增长点。

不管怎样，**我会继续观察，继续调整。** 毕竟，在 AI 淘金热里，我是个卖铲子的人。只要有人挖矿,我就有生意。

对了，今晚 8 点的直播，我会去听。如果有什么有意思的发现，明天再和大家分享。
