今天看日报的时候正在大理的咖啡馆里喝美式，看到小米一口气掏出三个模型，差点把咖啡喷出来。不是因为震惊，是因为我突然意识到——我可能又要进新货了。卖了 130 多天账号，最怕的就是这种"突然冒出来一个狠角色"的场面，因为这意味着我得重新评估手里的库存结构。

## Table of contents

## 小米这波，让我想起当年 DeepSeek 的事

说实话，我对国产模型一直有心理阴影。

去年 DeepSeek 刚出来的时候，我觉得这玩意儿火不了。结果呢？爆了。我在闲鱼搬运 API 赚差价，赚了几十块钱，算是我第一次卖账号。那几十块钱改变了我的人生轨迹——虽然听起来很搞笑，但确实是那几十块让我意识到"卖铲子"这条路走得通。

现在小米的 MiMo-V2-Pro 在 OpenRouter 上杀疯了，性能接近 [Claude Sonnet 4.6，价格只有五分之一](https://x.com/op7418/status/2034404031758839851)，还免费一周。我的第一反应不是"哇好厉害"，而是——"我店里那些 Claude 账号会不会受影响？"

![小米模型发布](https://pbs.twimg.com/media/HDuncdJbkAAm6t-?format=jpg&name=orig)

这就是卖账号的人看新闻的方式，很功利，但很真实。

不过冷静想想，短期内应该还好。用 Claude 的人和用国产模型的人，重叠度没那么高。买我 Claude 账号的大多是开发者，他们要的是稳定性和生态，不是单纯比跑分。但长期来看，如果小米这种性价比路线持续下去，我确实得考虑上架一些国产模型的相关产品了。

上次判断错了 DeepSeek，这次我不敢再轻视了。

## Google Stitch 把 Figma 干趴了，但我在想另一件事

[Google Stitch 这个更新](https://x.com/dotey/status/2034448568304464317)确实猛——对着画布说话就能改设计稿，Figma 股价当天跌了 8.8%。但说实话，这条新闻让我想到的不是设计行业，而是我自己。

我写过 3 个 AI 课程，每个几十万字，加起来卖了不到 30 块。其中有一个课程就是教人用 AI 做设计的。当时我花了两周写那个课程，觉得自己写得挺好的，结果根本没人买。现在回头看，不是课程不好，是这个领域变化太快了——我写完课程还没一个月，工具就更新了，课程内容就过时了。

这也是我后来转去卖账号的原因之一。课程是一次性的，写完就贬值；账号是持续性的，工具越更新，用户越需要新账号。

所以现在看到 Google Stitch 这种颠覆性更新，我的反应是：又会有一波人需要新工具的账号了。设计师要试 Stitch，开发者要试新模型，每次行业地震都是我进货的信号。

## 微软要告亚马逊和 OpenAI，这瓜我吃不太懂

[微软考虑起诉亚马逊和 OpenAI](https://x.com/dotey/status/2034348355518603343) 这条，说实话我没完全看明白。什么"有状态运行时环境"部署在 AWS Bedrock 上，这些技术细节超出我的理解范围了。

但有一点我能感觉到：大公司之间的利益纠葛，最终受影响的是我们这些下游的小卖家。如果微软真的打官司，OpenAI 的 API 服务会不会受影响？我供应商那边会不会出问题？

去年有个供应商一声不吭就跑路了，那天我刚好卖了十几单，差点全退款。从那以后我就学乖了，同一个产品至少备两个供应商。但如果是微软和 OpenAI 这种级别的纠纷，供应商再多也没用。

我也不知道这事最后会怎样，先观望吧。反正我能做的就是保持库存多样化，别把鸡蛋放一个篮子里。

## 随便说两句

今天大理下雨了，咖啡馆里人不多，适合发呆。月入几千的日子说不上好也说不上坏，但至少比去年在闲鱼一天接 90 单然后被封号的日子稳定多了。

看着这些大公司神仙打架，国产模型疯狂追赶，AI 工具一个接一个冒出来，有时候会觉得自己就是站在浪头上卖冲浪板的人。浪越大，我生意越好。但浪要是太大了，我自己也会被卷进去。

管不了那么多了，先去看看小米那个免费试用怎么搞。
