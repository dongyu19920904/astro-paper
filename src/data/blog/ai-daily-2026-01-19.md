今天看完日报，第一反应不是"哇，又有新工具了"，而是突然想起去年闲鱼被封号的那七天。

那时候我刚开始卖 AI 账号，一天能出 90 单，感觉自己要起飞了。结果第二天早上醒来，闲鱼账号直接被扣 20 分，封禁 7 天。那七天我天天刷后台，看着订单消息进不来，钱赚不到，心态崩了。现在想想，那次被封反而是好事——逼着我把业务搬到了自己的网站 aivora.cn，不用再看平台脸色。

今天日报里有条新闻让我想起这事：Anthropic 发了个报告，说 AI 不是简单地"抢工作"，而是在重构工作内容。修 Bug 占了 Claude 使用量的 6%，排第一。这个数据挺有意思的，因为我卖的 Cursor Pro 换号器，用户最常问的就是"能不能帮我改代码"。说白了，大家不是想让 AI 替自己写代码，而是想让 AI 帮自己改代码——写代码是创造，改代码是体力活。

## 卖账号的人看 AI 新闻

我看 AI 新闻的角度和别人不太一样。别人可能关心"这个模型性能怎么样"，我关心的是"这个会不会影响我卖的产品"。

比如今天看到 Ollama 支持 Anthropic API 格式了，我第一反应是：卧槽，本地模型也能调 Claude 工具了？那我卖的 Claude 三合一账号会不会降价？想了想，应该不会——本地模型虽然免费，但配置门槛高，普通用户还是更愿意直接买个账号用。

再比如 AionUi 这个开源工具，GitHub 上已经 5539 个 Star 了。这玩意儿是免费、本地、开源的 Cowork 替代品，适用于 Gemini CLI、Claude Code、Codex 等一堆工具。我自己试了试，界面确实简洁，响应速度快到几乎无延迟。但说实话，作为一个卖账号的，我反而挺开心——开源工具越多，用户越多，我生意越好。因为大部分人懒得折腾，他们只想要一个"开箱即用"的账号。

## Skills 是什么？我也不太懂

今天日报里有篇文章，宝玉写的，讲 Skills 是短期红利还是长期壁垒。我看了两遍，大概懂了一半。

他说 AI 发展分三个阶段：第一阶段是 Chatbot + Prompt，第二阶段是 Agent + 上下文工程，现在是第三阶段 Agent + Skills。Skills 作为一种具体形式可能会被取代，但你在 Skills 上投入的学习和实践，会成为你的长期壁垒。

这段话让我想起我自己的经历。一年前我在闲鱼接单定制 AI 指令，9.9 一份，一份要写半天，不赚钱但学到很多。后来转卖 AI 指令合集，收入直线下滑。再后来 DeepSeek 爆火，我在闲鱼搬运 API 赚差价，赚了几十块——这是我第一次卖账号。

现在回头看，那些"不赚钱"的经历其实都是积累。我当年写 Prompt 的时候，学会了怎么和 AI 对话；后来做上下文工程的时候，学会了怎么让 AI 记住更多信息；现在做 Skills 的时候，学会了怎么让 AI 完成更复杂的任务。每一波技术浪潮的"短期红利"，都是下一波浪潮的入场券。

所以别纠结 Skills 会不会被取代，而是问自己：通过做 Skills，我能积累什么能力？这些能力在下一波浪潮里还有没有用？

## 我的疯狂梦想和今天的新闻

我有个疯狂的梦想：借 AI 实现长生不老。5 年目标。听起来像民科，但我是认真的。

今天日报里没有直接相关的新闻，但有条让我想起这事：Google 开源了一个 Python 库 langextract，用于从非结构化文本中提取结构化信息，具有精确的来源定位和交互式可视化功能。这个库的核心优势是：能精确定位提取信息的来源（不是瞎编的），还有交互式可视化功能（方便调试和验证）。

我在想，如果把这个库用在生命科学领域，能不能从海量论文里提取出有用的信息？比如哪些基因和衰老有关，哪些药物能延缓衰老，哪些实验结果是可靠的。当然，我承认自己是小白，这些想法可能很幼稚。但我觉得，AI 在生命科学领域的应用，可能比我们想象的更快。

## 今天没什么特别的想法

说实话，今天的日报有点无聊，没什么让我眼前一亮的。大部分新闻都是"某某工具更新了"、"某某库开源了"，看多了就麻木了。

但有条新闻让我笑了：ThinkPad 小红点的三款帽子。1997 年，IBM 给 ThinkPad 的小红点加了两款新帽子，成为三个：原版猫舌头（Cat Tongue）、软边帽（Soft Rim）、软圆顶帽（Soft Dome）。ThinkPad 出厂都会配，用户根据手感偏好随时换。

我第一次知道这个冷知识。原来小红点还有这么多讲究，难怪 ThinkPad 用户都对它爱不释手。这让我想起我自己卖账号的经历——有时候用户不是在买产品，而是在买一种"手感"。就像 ThinkPad 的小红点，技术上可能没什么特别的,但用户就是喜欢。

我卖的 Cursor Pro 换号器也是一样。技术上可能没什么特别的，但用户就是喜欢那种"无限续杯"的感觉。这种感觉是无法被取代的。
