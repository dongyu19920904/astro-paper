今天看到 Sam Altman 要搞开发者座谈会的消息，第一反应不是"哇 GPT-5 要来了"，而是"卧槽，如果 OpenAI 放大招，我手里的 Claude 账号会不会受影响"。卖账号卖久了，看新闻的角度都变了。

## Table of contents

## Sam Altman 这波操作让我想起去年的 DeepSeek

说实话，Sam Altman 这种"开放姿态"我是有点怀疑的。去年 DeepSeek 爆火的时候，我也是后知后觉——当时觉得这玩意儿火不了，结果被打脸打得啪啪响。

那会儿我在闲鱼搬运 DeepSeek API 赚差价，一单赚几毛钱，累死累活赚了几十块。但那是我第一次卖账号，算是入行的起点。现在回想起来，如果当时判断对了，早点囤货，可能现在就不是月入几千而是几万了。

所以这次 Sam Altman 说要搞"新一代工具"，我的态度是：**先观望，别冲动进货**。

上次 OpenAI 搞 GPT Store 的时候，我也是兴奋了一阵子，结果那玩意儿雷声大雨点小。现在谁还记得 GPT Store？

## JetBrains 这个"漏洞"让我有点焦虑

今天看到 JetBrains AI 订阅用户可以免费无限用 Codex 的消息，我第一反应是：**这会不会影响我卖 Cursor 账号的生意？**

说实话，我也不太懂这些工具之间的差异。Cursor、Windsurf、JetBrains AI、Augment……太多了，我自己都分不清哪个好。但作为卖家，我观察到的是：**用户其实也分不清**。

很多人买账号就是"听说这个好用"，然后买了，用两天，又听说另一个更好，又来问我有没有。

我有时候想，我是不是应该写个对比文章？但转念一想，我自己都是小白，写出来怕误导人。

不过有一点我是确定的：**越卷，我生意越好**。工具越多，用户越迷茫，越需要有人帮他们"试错"。我卖的不是账号，是试错成本。

## Skill 生态爆发这事，我有点看不懂

今天日报里好几条都在讲 Skill，什么 NotebookLM Skill、前端开发 Skill 组合、微信 MCP……

说实话，我没太看懂这些东西是干嘛的。

我知道 MCP 是 Claude 的一个协议，可以让 AI 调用外部工具。但具体怎么用、能干嘛，我是真的不太清楚。

我试过用 Claude Code 写点简单的脚本，体验还行，但也就停留在"能用"的程度。那些什么"Lighthouse 性能分从 21 飙到 90"的操作，对我来说就像看天书。

但我有个直觉：**这些东西会越来越重要**。

就像当年智能手机刚出来的时候，大家也不知道 App Store 能干嘛。现在呢？没有 App 的手机还能叫手机吗？

所以我打算找时间研究一下 Skill 和 MCP。不是为了自己用，是为了**知道我的客户在用什么**。卖铲子的人，总得知道矿工在挖什么矿吧。

## 今天在大理的生活

写到这里，突然想说点和日报无关的。

今天大理天气很好，我在洱海边的咖啡馆写这篇博客。旁边坐着一对情侣在拍照，咖啡馆老板在逗他的猫。

有时候我会想，我这种"卖 AI 账号"的生意，能持续多久？

AI 发展太快了。今天还在卖的东西，明天可能就没人要了。就像我之前卖 AI 指令合集，收入直线下滑，最后几乎归零。

但换个角度想，**正因为变化快，才有机会**。大公司反应慢，我这种小卖家反而能快速调整。

好了，不多想了。明天早上 8 点 Sam Altman 的直播，我可能会看一眼。如果有什么大新闻，我再来更新。
