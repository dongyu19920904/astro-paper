今天看到 OpenClaw 星标破 21 万的消息，第一反应不是"哇好厉害"，而是一阵后怕——我差点又错过一个趋势。

## Table of contents

## 技术老鸟的诅咒，我也中过

日报里有句话特别扎心："技术老鸟一眼看穿——原来就是用自然语言创建定时任务。"

这让我想起去年 DeepSeek 刚出来的时候。当时我在闲鱼搬运 API 赚差价，有个做开发的朋友跟我说："这不就是套壳吗？没什么技术含量。"我信了，觉得这玩意儿火不了多久。

结果呢？DeepSeek 直接爆了，我那几十块的差价生意也跟着水涨船高。虽然赚得不多，但让我明白一个道理：**技术复杂不复杂，和产品能不能火，是两码事**。

OpenClaw 也是这个逻辑。定时任务？cron job？这些东西存在几十年了。但把它包装成"会说话的管家"，让普通人用自然语言就能操作——这才是值钱的地方。

我卖账号这一年多，见过太多"技术很牛但没人用"的产品，也见过"技术一般但用户疯抢"的东西。Cursor 为什么卖得好？不是因为它底层模型多厉害，是因为它把"AI 写代码"这件事做得足够傻瓜。

## Self-host 这波，我有点心动

今天日报里另一条让我琢磨了很久：Self-host + Agent 组合。

说实话，我之前对 self-host 没什么感觉。搭建麻烦、维护麻烦、出了问题还得自己修——我一个卖账号的，哪有这功夫？

但如果 Agent 能把这些活儿都干了呢？

我现在卖的账号，本质上是在帮用户省事——不用自己注册、不用自己充值、不用担心封号。如果未来 Agent 能让 self-host 变得像"插上电就能用"一样简单，那我卖的这些云服务账号，会不会慢慢没人要了？

这个想法让我有点焦虑，但又有点兴奋。焦虑是因为可能影响生意，兴奋是因为——如果真到了那一天，我自己也能用上更便宜、更私密的 AI 服务。

当然，这个"那一天"可能还很远。我也不知道具体要多久。但 OpenClaw 21 万星标这个数字，至少说明方向是对的。

## 说点和日报无关的

最近大理天气不错，下午经常去洱海边发呆。有时候会想，我这个"卖铲子"的生意还能做多久？

AI 工具更新太快了。去年这时候 Cursor 还是小众产品，现在已经是我店里的主力。谁知道明年又会冒出什么新东西？

不过想想也还好。工具会变，但"帮用户省事"这个需求不会变。只要我能跟上节奏，总有铲子可以卖。

今天就这样吧，没什么特别深刻的观点，就是看完日报随便想了想。
