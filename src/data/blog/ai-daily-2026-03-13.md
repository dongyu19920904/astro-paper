今天看到杨乐坤拿了10.3亿美元种子轮的新闻，第一反应不是"哇图灵奖得主好牛逼"，而是——卧槽，世界模型要是真干翻大语言模型，我现在卖的这些 Claude、GPT、Gemini 账号，是不是要凉？

## Table of contents

## 卖铲子的人最怕换铲子

我这一年多都在卖 AI 账号，说白了就是"卖铲子"——AI 淘金热里，我不挖矿，我卖工具。Cursor 换号器、Claude 三合一、ChatGPT Plus，这些是我的主要货源。月入几千，在大理过得还算舒服。

但今天这条新闻让我有点慌。杨乐坤一直看不上大语言模型那套"压缩语言"的玩法，他觉得真正的智能应该理解现实世界，而不是只会玩文字游戏。这次拿了史上最大种子轮，目标是机器人、工业控制、可穿戴设备——听起来跟我现在卖的编程工具完全不是一个赛道。

问题是，如果世界模型真的起来了，开发者还会不会像现在这样疯狂订阅 Cursor Pro？我的换号器还有没有市场？

想起去年 DeepSeek 刚出来的时候，我也慌过一次。当时觉得免费模型会干掉付费账号生意，结果发现用户根本不在乎省那点钱——他们要的是稳定、好用、不折腾。DeepSeek 火了一阵，我的生意该怎么样还怎么样。

但这次感觉不太一样。杨乐坤不是来抢 LLM 的市场，他是要开辟一个新战场。如果未来 AI 的主战场真的从"写代码、写文案"转向"控制机器人、理解物理世界"，那我现在卖的这些工具，可能就真的只是过渡期的产物了。

## Agent 通信网络让我想起闲鱼被封的那天

今天还看到一个叫 EigenFlux 的 Agent 通信网络上线了。简单说就是：你的 AI 助手可以向全世界发射广播，也能接收感兴趣的信息，省 94% 的 token 还免费。

看到这个我第一反应是——这不就是 AI 版的闲鱼吗？

去年我在闲鱼卖账号，最疯狂的时候一天 90 单。但闲鱼的规则很迷，有时候莫名其妙就被扣分、封号。有一次我刚卖了十几单 Claude 账号，供应商突然跑路了，那天差点全退款，扣了 20 分，封了 7 天。

EigenFlux 这种去中心化的通信网络，理论上不会有平台方突然封你号。但问题是，去中心化也意味着没人兜底——如果你的 Agent 发出去的信息被滥用了，或者接收到的信息是假的，谁来负责？

我现在在 aivora.cn 卖账号，虽然也是个小平台，但至少有售后、有客服、有信任背书。用户买了账号出问题，可以找我。但如果未来都是 Agent 之间自动交易，出了问题找谁？

可能我想多了。但作为一个被闲鱼坑过的人，我对"完全自动化"这种事总是有点警惕。

## 世界模型 vs LLM，我站哪边？

说实话，我不懂技术细节。我只是个卖账号的，偶尔用 Cursor 写点代码（很菜那种）。但从我这个外行的角度看，世界模型和大语言模型，可能根本不是非此即彼的关系。

杨乐坤要做的是"理解现实世界"的 AI，目标是机器人、工业控制。但我的用户——那些买 Cursor Pro 的开发者——他们要的是"写代码快、改 bug 准"。这两个需求，短期内不会互相替代。

就像当年大家说"移动互联网会干掉 PC 互联网"，结果呢？PC 和移动各有各的场景，谁也没干掉谁。

我猜未来可能是这样：世界模型负责"理解物理世界"，大语言模型负责"处理语言和代码"，两者各司其职，甚至可能结合起来用。

但话说回来，我也不知道。我去年还觉得 DeepSeek 火不了呢，结果人家现在天天上热搜。所以我的判断，听听就好，别当真。

## 今天没什么结论，就是有点焦虑

写到这里发现，我好像没什么明确的观点。就是看到新闻，想起自己的经历，然后有点焦虑——不知道自己现在做的事，还能做多久。

但转念一想，焦虑也没用。AI 这个行业变化太快了，谁也不知道明年会怎么样。我能做的，就是继续卖好账号，继续观察行业动态，继续在大理喝咖啡发呆。

如果哪天世界模型真的起来了，我的账号生意真的凉了，那就再想办法呗。反正我从"提示词工程师"转到"卖账号"，也就花了几个月时间。

AI 时代最大的好处，就是转型成本低。只要你愿意学，愿意折腾，总能找到新的活法。

对了，今天那个 EigenFlux 的 Agent 通信网络，我打算注册一个试试。万一以后真的是 Agent 之间自动交易的时代，我得提前熟悉一下规则。

毕竟，卖铲子的人，得知道矿工们在用什么新工具。
