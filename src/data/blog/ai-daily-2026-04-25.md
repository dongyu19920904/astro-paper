2026年4月25日，GPT-5.5 开放 API，DeepSeek-V4 同日开源，1.6T 参数，还不收钱。两颗炸弹同一天引爆，朋友圈全是"AGI来了"。但我坐在大理的出租屋里，脑子里转的第一个念头是：我架子上那批 Claude 账号，还能卖多久？

## Table of contents

## 卖铲子的人，也会被新矿机吓到

说实话，我这种卖账号的，理论上应该是最不怕模型迭代的那种人。

模型越多，用户越需要账号，我生意越好——这个逻辑我跟自己说了很多遍，也跟客户说过。但今天这个阵仗有点不一样。

GPT-5.5 开放 API 这件事，我关注的不是它有多强。我关注的是：有研究员把实验扔给它跑了一整晚，早上醒来 sweep dashboard 已经跑完，全程没碰过一行代码。这种事情以前只在 demo 里见过，现在是真实发生的。

然后 DeepSeek-V4 同一天开源，1.6T 参数，推理时只激活 49B，100 万 token 上下文，还免费。梁文锋说"让全世界用上 1.6T 的模型还不收钱"，这句话以前听起来像口号，今天看起来不像了。

我当年觉得 DeepSeek 火不了。

是的，我说过这句话。2025 年初 DeepSeek 爆火那会儿，我在闲鱼搬运 API 赚差价，赚了几十块，然后觉得这东西热度过了就完了。结果你看，现在 V4 出来了，1.6T，开源，免费，还在继续。

我的判断力，一如既往地烂。

## 我真正在焦虑的事

不是模型强不强，是替代关系。

我现在卖的东西里，有一部分是"帮用户绕过注册门槛"——国内用户注册 ChatGPT 麻烦，支付麻烦，我帮他们搞定，收个服务费。这个逻辑成立的前提是：用户想用，但自己搞不定。

但如果 DeepSeek-V4 免费、开源、国内能直接用，用户还需要绕这个弯吗？

我想了一下，暂时还需要。因为 DeepSeek 再强，它不是 Claude，不是 GPT-5.5，不是 Gemini。有些用户就是要用那个特定的工具，不是因为它最强，是因为他们的工作流已经绑在上面了。Cursor 用户不会因为 DeepSeek 免费就换掉 Cursor，他们需要的是 Cursor 账号。

但我也不敢太乐观。这个行业变化太快，我每次觉得"暂时没事"，过两个月就会有新的变量出来。

## Cursor 多 Agent 这件事，我有点羡慕

[Cursor 3 上线了 /multitask](https://x.com/dotey/status/2047790274949132357)，多个子 Agent 可以并行异步跑了。

我用 Cursor 写代码，水平很菜，但我能感受到这个功能的价值。以前一个任务没跑完，下一个只能等，现在可以同时跑。对我这种线程永远超载的人来说，光是看到这个功能描述就有点羡慕。

我的日常是：客服、补货、上新、调试、写内容、回售后，全部交织在一起，随时被打断。我没有"并行 Agent"，我只有一个脑子，还经常死机。

有时候我会想，如果我的业务系统也能像 Cursor 这样——把几个任务扔出去，自己去跑，我来收结果——那该多好。但现实是，我的"系统"大部分还是我本人在手动执行。自动化这件事，我想了很久，一直没真正做起来，因为每次刚要开始搭，就被新的客服消息打断了。

这是个死循环，我知道，但暂时还没找到出口。

## 今天最让我意外的一条

是那个 PPT Skill。

[歸藏把十年设计经验压进一个 Skill 文件](https://m.okjike.com/originalPosts/69eaccc10c669b00d55b4e9d)，Claude 先问你 6 个问题，然后生成一套 PPT，最终是单文件 HTML，直接用。

![PPT Skill 生成效果](https://cdnv2.ruguoapp.com/Ft0zLKXXLIMLgPSOeun5bg2OMEnzv3.png?imageMogr2/auto-orient/thumbnail/300x2000%3E)

我看到截图的第一反应是：这不像 AI 做的。

然后我想到我写过的那三个 AI 课程。每个几十万字，加起来卖了不到 30 块。我当时觉得内容质量是核心竞争力，只要写得够好，就能卖出去。结果证明我错了，内容质量和销量之间的关系，远比我想象的复杂。

但这个 PPT Skill 不一样。它不是在卖内容，它是在卖一个可以复用的能力包。用户拿到的不是一份 PPT，是一个"随时能生成好看 PPT 的工具"。这个逻辑，比卖内容本身要扎实得多。

我现在做的账号生意，某种程度上也是这个逻辑——我卖的不是某个模型的答案，我卖的是"随时能用上这个模型"的能力。只是我一直没有把这件事说清楚，包括对自己。

今天算是想清楚了一点点。
