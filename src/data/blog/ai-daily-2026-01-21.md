## Table of contents

今天刷日报看到 SuperPowers 这个项目——一个给 Claude Code 做的插件，2.7 万星。说是能"逼着 AI 先思考再写代码"，像资深程序员带新人那种感觉。看到这个数字的时候我愣了一下，因为我卖的柴犬助手本质上也是在做类似的事情：让 AI 编程工具更好用。

但人家 2.7 万星，我的月销才几百单。差距有点大啊。

## 开源项目火了，卖账号的怎么想

说实话，每次看到这种爆火的开源项目，我心情都挺复杂的。

一方面，开源生态越繁荣，用AI 编程的人就越多，我生意理论上会更好。这个逻辑我想了很久，应该没问题——你想啊，SuperPowers 让 Claude Code 更好用了，更多人愿意用 Claude Code 了，但Claude 账号不好弄，就来找我了。

另一方面，我又有点焦虑。这些工具迭代太快了。今天 Claude Code 火，明天可能就是别的什么 Code。我囤的货、搭的换号器、写的教程，随时可能过时。

上个月就吃过亏。Windsurf 那波热度来得快去得也快，我进了一批货，结果到现在还没卖完。倒不是亏钱，就是占库存，看着烦。

## 微软也下场了，这事有点意思

今天还有条新闻：微软放出了 agent-lightning，一个 AI Agent 训练框架，1.1 万星起步。

我对这个倒没什么直接的感觉——Agent 训练这种事离我太远了，我连代码都写不利索。但微软下场这件事本身挺有意思的。

你看啊，现在这个局面：
- OpenAI 有自己的 Agents
- Anthropic 有 Claude Code
- Google 有 Gemini CLI，今天还开源了 langextract
- 微软现在也来了

四家巨头都在抢这块蛋糕。作为一个卖铲子的，我其实挺开心的——越卷越好，卷出来的产品越多，用户选择越多，需求越大，我生意越好。

但话说回来，这种"开心"也挺短视的。万一哪天这些巨头把账号问题解决了呢？比如 Cursor 要是做了个国内直连版本，我换号器不就废了吗？

算了，想太远了。先把眼前的生意做好吧。

## 这周我在折腾的一件事

和日报无关，但想记一下。

最近在研究怎么把我那个三合一账号（Claude/Gemini/Codex）的教程做得更傻瓜一点。现在很多客户买完不会用，来问我，我就得一个一个教。

问题是我自己也不是特别懂。比如 Codex 那个，我知道怎么登录、怎么调用，但要说深度玩法，我真不行。有时候客户问的问题我得先去搜一下才能答。

这种感觉挺奇怪的——我卖的东西，我自己不完全懂。但好像也不影响卖？客户要的是账号能用，不是要我给他上课。

不过我还是想把教程做好一点。不然售后太累了，影响心情。

## 今天的新闻里还有个有意思的

Google 的 langextract，能从乱七八糟的文本里自动提取结构化数据。2.3 万星。

这个我能理解用途——以前我整理客户信息的时候，就得手动从聊天记录里扒订单号、扒联系方式，烦死了。要是有个工具能自动干这活儿，确实省事。

但我估计我不会用。一是我技术不行，二是我现在订单量也没大到需要这种工具的程度。

等哪天日单破200 了再说吧。（笑）

## 最后

今天大理下雨，在咖啡馆待了一下午。刷完日报写完这篇，外面还在下。

AI 这个行业变化太快了，我有时候觉得自己跟不上，有时候又觉得跟不上也没关系——反正我是卖铲子的，只要有人挖矿，就有人需要铲子。

至于挖的是金子还是空气，那是矿工的事，不是我的事。
