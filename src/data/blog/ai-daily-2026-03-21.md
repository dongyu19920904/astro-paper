一个卖 AI 账号的人，看到 Cursor 被扒底裤的第一反应不是"开源信任危机"，而是想起了自己被供应商坑过的往事。顺便聊聊 Google 又在免费烧钱、Vibe Coding 混战、以及为什么我对"PUA Coding"那个项目特别反感。

## Table of contents

## Cursor 这事，我太熟悉了

今天最炸的消息毫无疑问是 [Cursor Composer 2 被扒出底层跑的是月之暗面的 Kimi K2.5](https://x.com/dotey/status/2035017845801689440)。开发者调试 API 的时候发现模型 ID 写着 `kimi-k2p5-rl-0317`，Moonshot 那边的人直接出来确认 tokenizer 一致，问 Cursor 为什么不遵守许可证。

![image](https://pbs.twimg.com/media/HD3V6xfXwAEdyJt?format=jpg&name=orig)

技术圈炸锅了，讨论开源信任、许可证合规什么的。

但我看到这条新闻的第一反应完全不是这些。

我想起的是去年我刚开始做 AI 账号的时候，有个供应商跟我说他的 Claude 账号是"官方渠道"，价格比市场低 30%。我信了，进了一批货，上架卖了十几单。结果第三天，客户陆续来找我——账号全被封了。

我去问供应商，人直接把我拉黑了。

那天我在大理的出租屋里对着电脑发了一个小时的呆，然后开始一单一单退款。那种感觉不是愤怒，是一种很闷的无力感——你以为你在卖一个东西，其实你连这个东西的底层是什么都不知道。

Cursor 这事本质上是一样的。你以为你在用 Cursor 自研的模型，结果底层是 Kimi K2.5 套了一层皮。Cursor 发布的时候只说"对基座模型做了持续预训练"，全程不提 Kimi。

这不叫技术选择，这叫信息不对称。

做生意久了你会发现，最怕的不是产品不好，最怕的是你不知道自己在卖什么。我现在挑供应商，第一件事就是问清楚线路来源。不是因为我懂技术，是因为我被坑怕了。

所以看到有人替 Cursor 辩护说"用开源模型做二次开发很正常"——是的，技术上没问题，但你倒是说啊。闷声用别人的模型，连许可证都不遵守，这跟我那个跑路的供应商有什么本质区别？

## Google 又在烧钱了，白嫖窗口抓紧

另一条让我注意的是 [Google AI Studio 上线了 Build 功能](https://x.com/Gorden_Sun/status/2035012908795715739)，直接杀进 v0 和 Bolt 的地盘。接了 Firebase，能搭带后端的完整应用。

免费的。

我对"免费"这个词现在有条件反射了。做发卡站这一百多天，我太清楚"免费"意味着什么了——意味着先让你用上瘾，然后收费，然后涨价。Google 在这方面的历史记录可以说是完美的。

但话说回来，对我这种人来说，白嫖窗口就是机会窗口。我去年用 Bolt 搭过一版发卡站的前端，效果凑合但够用。如果 Google 这个 Build 真的能一口气搞定前后端，那我下次改版可以试试——反正不要钱，试错成本为零。

Vibe Coding 这个赛道现在是真的卷。Cursor、v0、Bolt、Windsurf、Google AI Studio Build……我自己卖这些工具的账号，说实话有时候都分不清哪个好。客户问我"Cursor 和 Windsurf 选哪个"，我只能说"都试试"，因为我自己也没有确定答案。

不过从卖家角度讲，越卷越好。工具越多，用户越迷茫，越需要有人帮他们降低试错成本——这就是我卖账号的价值。你不用一年花几百刀去订阅一个可能不适合你的工具，先买个短期号试试，不好用换下一个。

## PUA Coding 拿了 8.9k Stars，我笑了

最后说一个让我特别有感触的：[PUA Coding 那个项目一周涨了 8.9k Stars](https://x.com/dotey/status/2035052659334389979)，核心就是在 prompt 里写"不许说做不到""验证后才能说完成"。

我一年前就是干这个的啊。

在闲鱼上 9.9 一份定制 AI 指令，本质上就是帮客户写各种花里胡哨的 prompt。有人让我写"严厉老师型"，有人要"不许偷懒型"，有人甚至要"骂 AI 型"。我当时一份写半天，累得要死，赚不了几块钱。

但这段经历让我明白一件事：大部分人使用 AI 的瓶颈不是 AI 不够强，是自己不会拆任务、不会给上下文。贴一张"不许偷懒"的符就指望 AI 开窍，跟我去年写的那些 9.9 的指令合集一样——看着有用，其实是安慰剂。

真正有用的是什么？是像 [gstack](https://x.com/Gorden_Sun/status/2034937498020061486) 那样把流程结构化，是像 [LibTV 的 Skills 系统](https://m.okjike.com/originalPosts/69bbc8b025bae5661216c030)那样让 Agent 能接入工作流。不是对 AI 发脾气，是给 AI 搭脚手架。

8.9k Stars 只说明一件事：一年过去了，大部分人还停在我当年卖 9.9 指令的阶段。

## 一点碎碎念

今天大理下雨了，在屋里处理了一上午售后。有个客户买了 Cursor Pro 换号器，说用不了，我远程帮他排查了半小时，最后发现是他 VPN 没开。

这种事每天都在发生。我有时候会想，我到底是在卖 AI 工具，还是在做 AI 客服？

但转念一想，这就是"卖铲子"的代价。淘金热里卖铲子的人，不只是把铲子递过去就完了，你还得教人怎么握铲子、往哪里挖。

行吧，继续挖。
