今天看到 OpenClaw 创始人被 Sam Altman 挖走的消息，第一反应不是"哇好厉害"，而是想起了去年我一个供应商跑路的事。那天我刚卖了十几单 Claude 账号，结果供应商微信直接把我删了，库存全没了，差点全退款。

## Table of contents

## 开源项目换人这事，我见多了

OpenClaw 18 万星，创始人被挖走，项目移交基金会继续开源。听起来很美好对吧？

但我卖账号这一年多，见过太多"项目换人"的故事了。有些开源工具，作者一走，社区就散了。有些商业项目，核心团队被挖，产品直接烂尾。

说实话，我不太信"基金会继续开源"这种话。不是说 OpenAI 会食言，而是——一个项目的灵魂人物走了，剩下的人能不能撑起来，真的很难说。

这让我想起我卖的那些 AI 工具账号。Cursor、Windsurf、Augment... 每个背后都是一个团队。如果哪天核心开发者被挖走了，我手里的库存会不会变成废纸？

这种焦虑感，做这行的人应该都懂。

## GitHub 白嫖 30 天 Copilot Pro，我要不要进点货？

今天另一条让我眼前一亮的是 GitHub 悄悄上线了 Agents 入口，能免费领 30 天 Copilot Pro。

作为卖账号的，我第一反应是：这会不会影响我的生意？

想了想，应该不会。30 天试用期一过，该买的还是会买。而且很多人根本不知道有这个入口，或者懒得折腾。

但这也提醒我一件事：大厂的免费策略越来越激进了。GitHub 送 30 天，Google 送 Gemini 家庭组，OpenAI 也在搞各种优惠。

我这种"卖铲子的人"，夹在大厂和用户中间，利润空间会越来越小。

不过话说回来，越卷用户越多，我生意反而可能更好？我也不知道，走一步看一步吧。

## Gemini 3 "降智"这事，我有点幸灾乐祸

今天日报里好几条都在吐槽 Gemini 3 降智：上下文能力缩水、答非所问、莫名其妙的回复...

说实话，我有点幸灾乐祸。

不是我对 Google 有意见，而是——我之前判断错过一次。去年 Gemini 2.5 出来的时候，我觉得它会干掉 Claude，结果没有。现在 Gemini 3 出来，大家都说它强，我又开始焦虑我的 Claude 账号会不会卖不动了。

结果呢？Gemini 3 自己先翻车了。

这让我想起 DeepSeek 刚出来的时候，我觉得它火不了，结果它爆了。后来我在闲鱼搬运 API 赚差价，赚了几十块——虽然不多，但那是我第一次卖账号。

所以现在我学乖了：不预测，只观察。哪个好卖就卖哪个，用户用脚投票比我判断准多了。

## 今天的随想

在大理待久了，对这些新闻的敏感度好像在下降。

以前看到"18 万星项目创始人被挖"这种消息，我会兴奋半天，想着这意味着什么、我该怎么布局。

现在就觉得：哦，又一个人被挖走了。项目会继续，或者不会。我的账号该卖还是卖。

不知道这是成熟了，还是麻木了。

明天继续看日报，继续卖账号。大理的天气很好，咖啡馆的 WiFi 很稳定。

这就是我的生活。
