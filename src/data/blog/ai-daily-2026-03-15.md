今天看到 Claude 百万上下文正式开放的消息，第一反应不是"哇技术真牛"，而是——我那批 Claude 账号会不会因为这个涨价？然后又想，xAI 那边都快散伙了，马斯克从 Cursor 挖人，Cursor 年化收入 20 亿美元，我是不是该多进点 Cursor 的货？卖账号这一年多，我发现自己看新闻的角度已经彻底变了。

## Table of contents

## 百万上下文听起来很牛，但我更关心价格

Claude Opus 4.6 和 Sonnet 4.6 现在支持百万 token 上下文了，而且不加价。单次能塞 600 页 PDF，召回率 78%，碾压 GPT-5.4 的 36%。这数据确实漂亮。

但作为一个卖账号的，我第一时间想的是：这会不会影响我的定价？

去年 DeepSeek 爆火的时候，我在闲鱼搬运 API 赚差价，结果没几天价格就崩了，因为太多人涌进来卖。那次我学到一个教训——技术进步对用户是好事，对卖家不一定。

不过这次好像不太一样。百万上下文虽然开放了,但只有 Max、Team 用户才能用，门槛还在。而且国内用户注册 Claude 本来就麻烦，支付更麻烦。所以短期内，我这边的 Claude 账号需求应该不会降。

说实话，我自己也用 Claude 写过东西，体验确实比 DeepSeek 好。但我是外行，说不出具体好在哪里，就是感觉"更懂我想要什么"。可能这就是为什么 Claude 账号一直好卖吧。

## xAI 快散伙了，Cursor 却在狂飙

今天另一条新闻更劲爆：xAI 的 12 位联合创始人现在只剩 2 个了。马斯克自己都承认"一开始就没建对"。

这事儿让我想起去年我有个供应商，突然就联系不上了，那天我刚好卖了十几单，差点全退款。后来才知道他被封号了，直接跑路。

xAI 虽然不至于跑路，但这种大换血，说明他们的 AI 编程产品确实不行。马斯克从 Cursor 挖了两个高管过去，Cursor 现在年化收入 20 亿美元，是 AI 编程赛道最火的公司。

我卖的产品里，Cursor Pro 换号器一直是爆款。用户反馈说 Cursor 比其他工具"更聪明"，写代码的时候能猜到你下一步想干什么。我自己也试过，虽然我代码水平很菜，但确实感觉 Cursor 比较顺手。

现在看到 xAI 这个状况，我觉得我当初选择主推 Cursor 是对的。有时候卖账号也得看准赛道，不能什么火卖什么。

## Chrome 146 让 Agent 直接操控浏览器，这意味着什么?

Chrome 146 现在原生支持 WebMCP 了，Agent 可以直接操控浏览器，不需要第三方扩展。

这个我其实看不太懂技术细节，但直觉告诉我——这可能会改变很多东西。

以前让 AI 操控浏览器，要么启动单独的 Chrome 实例，要么装扩展，现在直接内置了。这意味着门槛降低了，更多人能用上这个功能。

对我来说，这可能意味着未来会有更多 AI 工具需要浏览器支持。我得关注一下，看看有没有新的账号需求出现。

说实话，AI 这个行业变化太快了。去年我还在闲鱼接单写提示词，9.9 一份，写半天不赚钱。现在卖账号，虽然也不算大富大贵,但至少稳定多了。

## 在大理卖账号的日子

扯远了。最近在大理的生活挺平淡的，每天起床第一件事就是看看有没有新订单，然后处理售后。

有时候会想，我这算不算"AI 淘金热里卖铲子的人"？别人用 AI 写代码、做项目，我就卖工具给他们。

前几天有个客户问我："你自己用这些工具吗？"我说用啊，但我不是专家，只是个普通用户。他说："那你怎么知道哪个好？"我说："我不知道，但我知道哪个卖得好。"

这就是我的生意逻辑——我不需要成为技术专家，我只需要知道用户需要什么。

今天看完这些新闻，我又在想，接下来该进点什么货。Claude 百万上下文开放了，Cursor 还在狂飙，xAI 快散了。这个行业真的是一天一个样。

不过也挺好的，至少不无聊。
