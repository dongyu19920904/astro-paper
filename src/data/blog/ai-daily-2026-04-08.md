今天最大的新闻是 Anthropic 年化收入 300 亿，超过了 OpenAI。Claude Mythos 数学竞赛题从 42% 干到 97.6%，代码修复能力全线碾压。但我看完这些数字，脑子里转的不是"AI 要改变世界"，而是：我闲鱼上那批 Claude 账号，今晚要不要调一下价？

## Table of contents

## 300 亿这个数字，对我意味着什么

说实话，我对 300 亿这个数字没什么感觉。

不是因为不震撼，是因为我每天盯的是另一套数字——今天出了几单，退款几笔，供应商那边货还够不够。Anthropic 赚多少钱，和我的关系是间接的：他们越赚越多，说明 Claude 用户越来越多，说明我这边的需求也在涨。就这样。

但有一条新闻让我真的停下来想了一下——

> "实际上 AI 培训只需要教一件事：如何稳定订阅上 Claude"

这条推文说的是，有人在课上认真讲提示词、讲工作流、讲 Agent 原理，结果学员最后问的是："老师，Claude 怎么付款？"

我看到这里笑了。因为这就是我每天在做的事。

我不是在教人用 AI，我是在帮人跨过那道"怎么付款"的门槛。Anthropic 收入 300 亿，财富 10 强 8 家是客户，但对普通用户来说，最高的门槛可能就是那张信用卡，和那个不知道从哪注册的账号。

我卖的就是这个。

## 那个"Claude 接微信"的机会，我今天认真想了想

商机那边提到一个项目：Claude-to-IM-skill，已经有人跑通了，可以把 Claude 接进微信群。

我的第一反应是：这个能卖。

不是因为技术多厉害，而是因为我太了解那个场景了——用户看到"Claude 可以在微信里帮你干活"这个画面，心动了。然后点进 GitHub，看到一堆英文命令，直接关掉。

这个关掉的瞬间，就是我的生意。

以前我在闲鱼卖 DeepSeek API 的时候，也是这个逻辑。技术上没什么难的，就是帮人省掉那两小时折腾。那时候一天能出几十单，后来被封号，扣了 20 分，停了 7 天。但那段时间让我明白一件事：**需求不是你创造的，需求是已经在那里的，你只是站在对的位置上接住它。**

Claude 接微信这个，今天刚被大量转发，正是心动人最多的时候。我打算今晚先自己装一遍，截一张 Claude 在微信里回复的图。这张图就是商品封面，比任何文案都好使。

## Mythos 用不上，但我不焦虑

Claude Mythos 现在只开放给 Project Glasswing 的 12 家合作伙伴，苹果、微软、亚马逊那种级别的。普通用户用不上，我也用不上。

以前我可能会焦虑——最强的模型我没有，我卖的东西会不会被淘汰？

现在不会了。

因为我发现一个规律：每次有新模型出来，不管多强，普通用户的第一个问题永远不是"这个模型能做什么"，而是"我怎么用上它"。Mythos 开放给普通用户可能是 Q2 到 Q3 的事，到时候又是一波需求。

我只需要在那个时间点上，手里有货就行。

现在能做的，就是把现有的 Claude 账号卖好，把配置说明包做扎实，等 Mythos 开放的时候，我已经有了一批信任我的老客户。

## 有一件事我一直没想清楚

今天看到 MemPalace，说是 30 倍无损压缩，能给 AI 一个真正的"记忆宫殿"。完全本地运行，不调外部 API。

技术上听起来很厉害。但我盯着这条新闻看了很久，没想明白一件事：**普通用户真的在意 AI 记不记得他们吗？**

我问过几个买家，他们说的最多的不是"AI 忘了我说过什么"，而是"AI 回答不够准"、"不知道怎么问"。记忆问题，好像是技术圈更在意的事。

当然我可能判断错了。我当年觉得 DeepSeek 火不了，结果它爆了，我在闲鱼搬运 API 赚了第一桶金（虽然只有几十块）。

所以 MemPalace 这个，我先观望。等中文社区跑出来真实反馈，再决定要不要做成产品。

今天先把 Claude 接微信这件事跑通。一步一步来。
