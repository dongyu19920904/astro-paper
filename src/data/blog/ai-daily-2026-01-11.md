今天看到 Tailwind CSS 创始人自曝裁员 75%、收入暴跌 80%，第一反应不是"哇好惨"，而是——我懂这种感觉。

去年我写过 3 个 AI 课程，每个几十万字，加起来卖了不到 30 块。是的，30 块。不是 30 万，是 30 块人民币。

当时我还挺骄傲的，觉得自己写的东西有价值，内容扎实，案例丰富。结果呢？DeepSeek 一出来，免费的、更新更快的教程满天飞，我那些"精心打磨"的课程瞬间变成了废纸。

Tailwind 的遭遇本质上是一样的：**你的知识被 AI 吃掉了，而且吃得比你自己消化得还快。**

以前开发者查 Tailwind 文档，顺便看到付费产品 Tailwind UI，这是他们的商业模式。现在呢？Cursor 直接把代码写好了，谁还去翻文档？

我卖课程也是这个逻辑——以前大家不懂 AI，愿意花钱学。现在随便问问 Claude，它教得比我好，还不收钱。

![Tailwind CSS 收入暴跌](https://pbs.twimg.com/media/G-UbBr7bMAAuGWT?format=jpg&name=orig)

---

说到这个，今天日报里还有一条让我挺在意的：[Anthropic 全面封堵反代通道](https://x.com/op7418/status/2009984648965943609)。

作为一个卖 AI 账号的，这种消息我看到就会条件反射地想：我的供应商那边会不会出问题？

Claude 账号一直是我店里的热门产品，但 Anthropic 这家公司真的很"轴"。之前就封过一波，这次连 XAI 和 Open Code 的反代都堵了。新装的 Claude Code 想用第三方 API？得手动加参数才能绕过登录。

有意思的是，OpenAI 的策略完全相反——[他们官方拥抱 Open Code](https://x.com/op7418/status/2010031968399028555)，Plus/Pro 用户可以直接授权使用。

两家公司的画风差太多了：
- Anthropic：筑墙、封堵、防薅羊毛
- OpenAI：拆墙、开放、以生态换市场

从我卖账号的角度看，OpenAI 这种策略对我更友好。但说实话，我也不知道哪种策略长期来看更聪明。Anthropic 可能是被薅怕了，毕竟 Google Antigravity 这次也顶不住了，[Pro 用户开始限额](https://x.com/op7418/status/2009982977510658177)。

---

今天日报里还有一条让我想了很久：[中国 AI 圆桌实录](https://m.okjike.com/originalPosts/6962523f5415148f2b15d07a)。

阿里的林俊旸说了句大实话："OpenAI 们的算力比我们大 1-2 个数量级，人家在'浪费'算力做探索，我们光应付业务就亮红灯了。"

主持人问：3-5 年后中国 AI 领先的概率？答案是 20%。

但唐杰老师说了句话触动我："如果我们笨笨地坚持，也许走到最后的就是我们。"

这句话让我想起自己。

我从闲鱼 9.9 一份的提示词开始做，一份要写半天，不赚钱。后来转卖指令合集，收入直线下滑。再后来搬运 DeepSeek API 赚差价，赚了几十块。写课程血亏。被闲鱼封号。

但我还在做。现在月入稳定几千，在大理过着半躺平的生活。

不是因为我聪明，恰恰相反，我经常判断错误。当年我觉得 DeepSeek 火不了，结果它爆了。我觉得卖课程能赚钱，结果血亏。

但我"笨笨地坚持"了下来。

也许这就是唐杰老师说的那种状态吧。不是因为看清了未来，而是因为没有别的选择，只能一直做下去。

---

最后说个和日报无关的事。

大理最近降温了，咖啡馆里的人明显少了。我每天的工作流程是：早上处理订单，下午看日报写东西，晚上发呆。

有时候会想，我这个"卖铲子的人"能做多久？AI 工具越来越便宜，免费的越来越多，我的生意会不会像 Tailwind 一样被吃掉？

不知道。

但今天先把这篇写完，明天的事明天再说。
