这两天国内大模型像是商量好了一起动手。豆包上线任务模式，MiniMax M3 宣布永久半价，GLM-5.2 开源，元宝接入 ima 知识库——同一天，四件事。我看完的第一反应不是"AI 真厉害"，是打开了自己的后台，把在卖的那几个 Claude 中转和三合一账号价格看了一遍。

## Table of contents

## 价格战这一刀，切到我这里是什么感觉

MiniMax M3 这次降价的幅度不小。输入 ¥2.1，输出 ¥8.4，比 GLM-5.1 便宜到只剩三分之一，比 K2.6 便宜六七成。说实话，API 价格战跟我卖的成品账号不是同一个市场，但我一直觉得这两个东西是联动的——API 便宜了，用 API 套壳的产品就更多，用 API 自己跑业务的人就更多，买我这种"一号多用、省心托管"成品账号的人就会……往哪个方向变，我还真没想清楚。

有可能是变少：会折腾的人直接上 API，不买我的了。也有可能是变多：API 越多越乱，普通人越搞不明白，反而来买现成的。我现在倾向于后者，但历史上我判断错的次数也不少，说出来仅供参考。

DeepSeek 那边缓存读取 ¥0.025，M3 是 ¥0.42，差了将近 17 倍。这个差距我其实没能力评价技术面，但从卖账号这个角度看，DeepSeek 的生态优势在国内一直被低估——主要是用的人太多，各种套壳和教程都是围着它转的，形成了一个惯性。M3 再便宜，要撼动这个惯性，得有一段时间。

## 豆包那个"任务模式"，我觉得还早

豆包上线 Agent 功能，自动搜索、自动做 PPT、全链路跑起来。这条新闻我看了两遍，第一遍想的是"这玩意儿能替代我用 Cursor 写脚本吗"，第二遍想的是"这会不会变成一个我该卖账号的产品"。

暂时都不是。原因很简单——我现在手上的客户，大部分人用 AI 还停留在"聊天问答"阶段，能把 Cursor 用起来的已经算进阶用户了。豆包 Agent 这种全链路自动化，他们大概率打开用一次，然后就不知道下一步该干什么。

这不是说豆包 Agent 不行，是普通用户还没到那个使用习惯。我卖账号 130 多天了，接触的用户少说几百个，感受最深的一件事是：真正的门槛不在账号，在"第一次用起来"。AI 工具越强大，这个门槛反而越高，因为功能多了，用户更不知道从哪入手。这也是我客服压力最大的来源——不是退款纠纷，是"我买了不会用"。

所以豆包 Agent 上线，我短期不会去跟，继续看。

## 那条"Claude 帮我发了半年资讯"让我挺有共鸣

有人让 Claude 接管 AI 资讯的初稿，自己只做微调。他说你平时看到他发的那些，大部分都是 Claude 先写、他改一下就发了。

我其实也在做类似的事，只是规模小得多。我的日报内容现在有一部分会丢给 AI 先跑一遍，自己再过一道。不是偷懒，是真的撑不住——每天客服、补货、上新、售后、教程，已经把时间切成碎片，再要我从头到尾每篇全手写，内容就直接断更了。

但我也注意到一个问题：用 AI 写久了，某些内容会开始有一种"AI 味"——句式太整齐，结构太工整，读起来像模板。我有时候看自己写的东西，会隐约觉得"这段不像是我说话"，但又说不清楚哪里不对。这种感觉我不太喜欢，所以现在的做法是：AI 出框架和素材整理，关键的判断和那几句"没人告诉我怎么说"的话，必须自己写。

算不算解法我不知道，至少到现在没有读者来问"你这个是 AI 写的吧"。

## 顺手说一下那个 AI 炒股的事

有人用 Codex 装了个开源 Skill 盯盘，AI 给指令说高开卖中天科技，本人没听，股票跌了 3%，AI 赢了。

这条我看了想笑，但笑完有点不舒服。不是因为 AI 赢了——而是因为这种"AI 说得对、人没执行、结果赔了"的故事，很容易被反复引用成"以后要听 AI 的"的论据。但开源 Skill 做出来的信号，赢一次真的说明不了什么。我在闲鱼卖账号最早期，也见过用户来问"能不能让 AI 帮我炒股"，每次我都是同一句话：你自己都不知道它的胜率，凭什么赌？

工具已经能出牌，但牌面对不对，还是得自己判断。这个责任没法转移给 AI，现在不行，以后也不行。
