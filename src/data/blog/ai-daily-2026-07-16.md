## 开源砸进来的那一刻，我在算账

Mira Murati 把 Inkling 扔出来的时候，我正在处理一个售后——有个用户说 Claude 三合一用不了，让我给他换号。

我一边给他排查，一边瞄了一眼新闻。看到"1T 参数开源多模态"这几个字，第一反应是：妈，这东西要是跑分真能打，我那些闭源账号的逻辑是不是得重新想一遍？

跑分一看：HLE 46.0%、SWE-bench 77.6%、GPQA 87.2%，[和 DeepSeek V4 Pro 同档](https://x.com/Gorden_Sun/status/2077523795167129852)，Apache 2.0 随便用。

我愣了大概三秒。

然后我意识到，我其实早就应该预料到这天会来。DeepSeek 爆出来那次我就没反应过来，坐等了两周才动。这次好歹没慌太久。

但有个问题我没想清楚：开源模型越来越强，对我这个卖账号的，到底是利好还是利空？

理论上是利空——模型强了、开源了、免费了，用户还需要买账号干嘛？

但现实是，每次模型大爆发，我的订单都在涨，不是跌。

DeepSeek 爆火那次，我在闲鱼搬运 API 赚差价，第一次感受到"模型越火，流量越大"是什么意思。后来 Claude 3.5 出来，Cursor 销量跳了一波；GPT-4o 免费开放，反而带来了一批"尝到甜头想买更好的"用户。

所以我现在的判断是：开源不是我的敌人，能用但不方便用，才是我的市场。

Inkling 1T 参数，你能跑吗？消费级 GPU 跑 MoE 激活 41B，已经不算轻松。大多数人没那个硬件，也没那个时间去折腾。他们需要的，还是一个拿来就能用的入口。

这个入口，就是我在卖的东西。

## 苹果 AI 过审这件事，我等得比用户还急

国行苹果 AI [终于通过备案接入千问](https://www.36kr.com/p/3896830991320962)了。7 款手机端侧 AI 一次性过审，阿里千问进了 Apple 智能。

说实话，我对这个新闻的情绪挺复杂的。

一方面，我是苹果用户，等这个功能等了两年，终于要来了。

另一方面，我卖账号，这件事对我是个提醒：手机原生 AI 铺开的速度，比我想象中快。

以前用户来问我买账号，一大半是因为"没有渠道用到好的 AI"。等苹果 AI 全量推送，iPhone 用户直接系统级可以用，那部分"只是想试试"的轻度用户，就不需要来我这里了。

不过我估摸了一下，真正会被挤掉的，不是我的核心用户。

来买我家东西的，基本分两种：一种是有具体需求的，比如要跑 Cursor Pro 的、要用 Claude Code 的、要大量调用 API 的；另一种是觉得直接买比自己注册省事的。第二种会流失一部分，第一种不会。

苹果 AI 能帮你写邮件、总结备忘录，但它替代不了 Cursor 写代码，替代不了 Claude 长上下文分析，替代不了 Codex 跑 Agent 任务。

只是，这个趋势让我警觉：我得把商品结构往"需要专门入口才能用的刚需工具"方向推，而不是继续卖那种"聊天用的 ChatGPT 普通号"——那块阵地迟早会被各家手机厂商的系统 AI 蚕食掉。

## 每天一千块利润，我最焦虑的事不是竞争

有时候朋友问我，你现在一天能挣多少，我说大概一千。他们都觉得挺好。

但我知道，这一千块利润背后是什么：每天从早到晚弹出来的消息，用户说"我用不了"、"这个怎么设置"、"你给的号怎么登录不上"、"能退款吗"……

今天 Inkling 出来，我一边感叹开源进展神速，一边心里有个声音：这玩意儿又要有人来问我"能不能买这个"了。

我还没搭好 AI 客服，一直在找合适的方案，但没一个让我觉得"稳，能用"。要么答错，要么格式乱，要么用户一反问就穿帮。我试过几个，最后都没撑过一个月就切回手动了。

所以，开源模型越来越强这件事，对我的最大意义，其实不是"又有新货可以卖了"，而是：**有没有哪个够强的本地模型，能让我把售后客服这件事真正自动化掉？**

Bonsai 27B 能在 iPhone 上跑，这个消息我看了两遍。[1-bit 版 3.9GB，保留 FP16 约 89.5% 的能力](https://x.com/Gorden_Sun/status/2077255908640313376)——这个参数，比我想象中更能打。

如果有一天，一个本地跑的小模型真的能稳定处理我的 FAQ、售后分诊、教程指引，我愿意花时间认真搭一遍。

不是为了炫技，是真的很累了。

## 我的一千个用户里，没有一个在折腾 1T 参数

写到这里我忽然想到一件事。

Inkling 出来之后，有很多人在讨论"开源终于打平闭源"、"AGI 提速"之类的。

我也觉得挺激动的，但我更清楚：我的一千个用户里，没有一个人现在会去下载 Inkling 跑起来。

他们大多数人，连 Cursor 的换号器怎么用都要问我好几遍。

这个落差，就是我存在的价值。

开源再强，也需要有人帮他们搭好，包装好，解释好，出了问题兜住。

这就是我在做的事。有时候觉得很琐碎，有时候觉得其实挺稳。

今天大理没下雨，难得。
