今天看完日报，最大的感受不是 GPT-5.4 有多牛，而是 OpenClaw 一天烧 25块这条新闻，让我想起了自己刚开始卖 DeepSeek API搬运差价的那段日子。那时候我觉得 API 调用成本低到可以忽略不计，结果真有客户跑来跟我说"我一天花了好几十块"，我当时还不信。现在看，Agent 时代的 token 消耗量，完全是另一个量级。

## Table of contents

##卖铲子的人也会被铲子价格吓到

先说 [OpenClaw 一天烧 2500 万 token](https://www.v2ex.com/t/1196365#reply0) 这件事。

有个开发者在自己服务器上跑 OpenClaw，接的是 MiniMax 的 M2.5，就让它干点简单活——配个Telegram Bot，写几个定时任务。结果半天 25 块没了。2500 万 token，200 多次请求。

这让我想起年初的一件事。有个客户从我这买了 Claude 账号，是个独立开发者，买了之后第三天来找我，说"你这号是不是有问题，怎么额度用得这么快"。我查了半天，最后发现他是把 Claude 接进了一个自动化工作流，每次触发都会带上完整的项目上下文。一天下来，token刷刷地跑。

号没问题，问题在于 Agent 模式下，AI 不是"你问一句它答一句"，而是"你说一句它思考十轮、调工具二十次"。

这对我的生意意味着什么？说实话，短期内是利好。因为用量越大，账号消耗越快，客户换号频率就越高——我的 Cursor 换号器和柴犬助手本来就是吃这个逻辑的。但长期看，如果 Agent 把成本拉到普通用户承受不了的程度，那用户干脆不用了，我也没生意。

所以我反而在盼着模型降价。越降越好。最好卷到地板价。

## GPT-5.4 百万上下文，然后呢？

[GPT-5.4](https://www.aibase.com/zh/news/26000) 确实是大新闻。百万 token 上下文窗口，金融法律跑分亮眼，一口气三个版本。

但我第一反应是——定价多少？

果然，贵。

![GPT-5.4](https://upload.chinaz.com/2026/0306/6390838957996076916065119.png)

作为一个卖 AI 账号的人，我观察到一个很明显的趋势：OpenAI 越来越走高端路线，但我的客户群体其实是"想用好工具但不想花太多钱"的那群人。他们来[aivora.cn](https://aivora.cn) 买账号，本身就是因为官方价格劝退。

现在 GPT-5.4 Pro 的定价更高了，这意味着什么？来我这买号的人只会更多。但同时，也有更多人会被推向 DeepSeek、MiniMax 这些国产替代品。

更讽刺的是 [洗车测试](https://www.v2ex.com/t/1196407#reply18)。GPT-5.4 Thinking 号称推理能力大升级，结果"洗车店离我 50 米，该走路还是开车？"这种问题还是答不对。

![洗车测试翻车](https://i.v2ex.co/ztQ0wmwn.png)

我当时看到这个测试，第一反应是笑了。第二反应是——等等，这其实挺吓人的。一个能处理百万 token 上下文、能在金融领域跑出高分的模型，居然不具备五岁小孩都有的常识。这说明"智能"和"聪明"真的是两回事。

我想起我自己犯过的类似错误。去年 DeepSeek R1 刚出来的时候，我觉得"就那样吧，火不了"。结果呢？闲鱼上 DeepSeek API搬运的单子直接爆了，我还是后知后觉跟进的，赚了几十块就没了。我的常识判断也经常翻车，跟 GPT-5.4 一样。

## 小米也下场了，手机 Agent 要卷起来了

[小米发布 Xiaomi miclaw](https://www.aibase.com/zh/news/26000)，一看就是跟风OpenClaw。

但这件事对我有点意义。我卖的账号主要是 PC 端的编程工具——Cursor、Claude、Gemini 这些。如果 Agent 的主战场真的转移到手机端，那我是不是也得开始关注手机端的 AI 工具账号？

现在还不急。邀请制内测说明产品还很早期，小米的 MiMo 大模型我也没用过，不知道水平怎么样。但方向是对的——AI 从"你打开一个对话框"变成"AI 直接替你操作手机"，这个趋势一旦成熟，对所有人的使用习惯都会是颠覆性的。

不过话说回来，那个[给OpenClaw 搞像素办公室](https://mp.weixin.qq.com/s?__biz=MzUxNjg4NDEzNA==&mid=2247531911&idx=1&sn=95d400dc01f9e4f6c7ebf8be985b58f9)的项目是真的好玩。AI 在后台跑任务的时候，变成一个像素小人在虚拟办公室里敲键盘、喝咖啡、面壁思过。说实话，我特别想给我的换号器也搞一个这样的界面——让客户能看到 AI 在后台"忙活"的过程，哪怕只是个动画。

## 今天的真实感受

坐在大理的咖啡馆里写这篇东西，外面下着小雨。

GPT-5.4 发布、小米下场、Agent烧钱……每天都有大新闻，但我的生活节奏其实没什么变化。早上起来看一遍日报，回几个客户消息，处理几个售后，下午发呆或者散步。

有时候我觉得自己挺幸运的——站在 AI 淘金热里卖铲子，不用去赌哪个模型最强，不用焦虑技术路线选错了。哪个火我就进哪个的货，就这么简单。

但有时候也焦虑。比如今天看到 [AI 破解 Dia 浏览器的Cookie 加密](https://x.com/vista8/status/2029971883811287277)，我就想——如果 AI 的安全攻击能力越来越强，会不会有一天连账号交易这个生意模式都被干掉？

想不明白。先不想了。

反正等等党永远不亏。今天的模型明天就降价，今天的工具明天就免费。我的策略是——别囤货，快进快出，赚到手的才是真的。今天的日报让我想起一个问题：AI 越来越能干，但谁来买单？

今天大理下了点小雨，我在屋里翻日报，翻着翻着就笑了——GPT-5.4 百万上下文窗口听起来牛得不行，结果连"洗车店在50 米外该走路还是开车"都答不上来。这种反差感太真实了，跟我卖账号的日常一模一样：客户觉得买了Pro 账号就能上天，结果发现 AI 连他的需求都理解不了。

## 百万上下文，然后呢？

说实话，看到 GPT-5.4 发布的时候，我第一反应不是"哇好厉害"，而是"定价多少"。

果然，贵。

我卖账号这一行，对价格极其敏感。每次OpenAI 涨价，我这边就得跟着调，然后就有客户来问"怎么又贵了"。我能怎么说？我也是被涨的那个啊。

但真正让我在意的不是价格本身，而是[洗车测试那条](https://www.v2ex.com/t/1196407#reply18)。

![GPT-5.4 Thinking 洗车测试翻车](https://i.v2ex.co/ztQ0wmwn.png)

50 米远的洗车店，走路还是开车？这种题我家楼下卖烧烤的大叔都不用想。但号称"推理能力大幅升级"的 GPT-5.4 Thinking 就是答不好。

这让我想起去年我写AI 课程的经历。我写了三个课程，加起来几十万字，花了无数个通宵，用AI辅助整理资料、润色语句。AI 在处理结构化内容的时候确实好用，但每次涉及到"这个例子放在这里合不合适""读者看到这里会不会觉得突兀"这种需要常识判断的地方，它就开始胡说八道。

最后那三个课程加起来卖了不到 30 块。

所以我现在对"模型又升级了"这种新闻，心态很平。能力确实在涨，但离"真正好用"中间还隔着一个叫"常识"的大坑。

## 小龙虾一天烧 25 块，这才是真问题

今天日报里最让我有感触的是[OpenClaw 烧钱那条](https://www.v2ex.com/t/1196365#reply0)。

一天 2500 万 token，25 块直接清零。还只是做了"配个Telegram Bot、写几个定时任务"这种小活儿。

我太理解这种感觉了。

去年我刚开始用Cursor写代码的时候（对，我会写一点点，很菜的那种），有一次忘了关 Agent 模式，让它自己跑了一晚上。第二天早上一看，额度直接见底。它在那儿反复改一个bug，改了又改回去，改回去又改，循环了几十次。

Agent 这个东西吧，理想很美好——"AI 替你干活"。但现实是，它干活的方式像一个特别勤奋但完全不懂节约的实习生。你让它订个会议室，它能把整个日历App翻个底朝天，顺便帮你查了天气、汇率和附近的餐厅。

[小米推出 Xiaomi miclaw](https://www.aibase.com/zh/news/26000) 也是同一个方向。手机端Agent，让AI 直接操控手机完成任务。方向肯定没问题，但——谁付这个 token钱？手机厂商补贴？用户自己掏？按现在的烧法，普通人真用不起。

我倒觉得，这波Agent 热潮最大的受益者可能不是 Agent 本身，而是模型厂商。大家都在拼命调用 API，token 消耗量比聊天场景高几十倍，模型厂商坐着收钱就好。

等等，那我卖的那些账号呢？

如果 Agent 真的普及了，用户对Pro 账号的需求会暴增还是暴跌？暴增是因为大家都想用高级模型跑 Agent；暴跌是因为烧不起，干脆回去手动操作。

我也不知道。真的不知道。

##阿里那条，我有点话想说

[林俊旸离开千问团队](https://m.okjike.com/originalPosts/69aabd0e50600952ffb7e176)那条新闻，分析写得挺犀利的——阿里在Coding、Agent、OpenClaw 几波浪潮里全部缺席。

我从卖家角度说个体感：我店里卖的账号，Claude、ChatGPT、Gemini 是大头，Cursor 换号器是爆款。通义千问的需求？基本为零。

不是说千问不好用，免费嘛，谁都能用，没人需要找我买。但问题是，免费也没见多少人真的深度用起来。我在闲鱼卖了一年多账号，接触了上千个客户，买Claude 的人知道自己要干嘛，买 Cursor 的人知道自己要写代码，买通义的人——没有这个品类。

生态这个东西，不是你模型能力到了就自动长出来的。你得让开发者觉得"用你的模型能赚到钱"或者"用你的模型能做到别人做不到的事"。MiniMax靠接入龙虾生态逆袭，这个路子就很聪明。
