## 我卖的是铲子，但铲子也在变

我在 aivora.cn 卖 AI 编程工具的账号，Kiro、Windsurf、Augment、Warp……前两年这些工具还算有门槛，普通人用不上，所以有人愿意买号。

现在 Raycast 推 Glaze，说一句需求，软件生成，内置 Store 一键装。有人用它调 Spotify API 做了自己的音乐电台，有人装了图片标注工具，有人做 Agent Skill 管理器——[都是真实案例，不是 demo](https://x.com/vista8/status/2063443048458264603)。

我卖铲子三年多，第一次感觉铲子本身也要变形了。

不是说 Kiro 或 Windsurf 明天就卖不动，而是——用户买账号的理由在变。以前是"这工具我想用但用不上"，以后可能是"我要做个东西，谁能帮我最快做出来"。这两个需求，表面上差不多，其实是两个不同的市场。

我卖的那批账号是前一种市场。后一种市场长什么样，我还没想清楚。

## Glaze 让我想起了闲鱼爆单那天

2025年初，DeepSeek 爆火，我在闲鱼搬运 API 赚差价。那段时间每天睡前刷消息，睁眼又看，生怕供应商断货。有一天真的爆单了，一天 90 单，我一个人盯着屏幕从早到晚发货链接，发到手软。

那个状态现在想起来，挺魔幻的——赚的不多，但感觉很亢奋，觉得自己踩在了一个浪头上。

Glaze 给我类似的感觉。不是那种"这东西真牛"的兴奋，而是"这个浪来了，我在哪儿"的焦虑兴奋混合体。

我现在的业务，发卡站、API 中转、各种账号，本质上是一个交付层——我不生产模型，我只是帮人用上模型。Glaze 在做的事，是把"交付层"往下压，直接压进用户需求那一层。以后用户有个小工具需求，直接说一句话，Glaze 给他做出来，他还需要买 Windsurf 的账号吗？

我真的不确定。但我也不打算现在就下结论，因为我经常判断错——当年我觉得 DeepSeek 火不了，然后那年我靠 DeepSeek API 差价赚的是这两年最开心的一笔钱。

## 今天真正让我有点在意的，是那条前端审美排名

[这份主观排名](https://x.com/vista8/status/2063264223879389431)，Claude Opus 4.8 排第一，然后是 kimi、GPT……没有 Benchmark，就是真实使用手感。

我自己卖 Claude 账号，所以我有点私利，但我觉得这个排名是真实的。我自己用 Claude 写过一些简单页面，风格确实比 DeepSeek 好看——不是技术层面的好看，是那种"排版有人味儿"的好看。不知道怎么描述，就是感觉它懂一点设计的基本审美。

这条新闻对我的业务是个好消息。Claude 账号我一直卖，如果它在前端这块的口碑越来越稳，拉新就更好拉了。

但我也知道，这种"口碑领先"很脆弱。今天排第一，三个月后 Kimi 或者 GPT 出一个更新，可能排名就换了。我做账号生意最怕的就是这个——绑定一个产品太深，然后那个产品掉了。

所以我一直在卖"合集"，不押单一产品。Claude/Gemini/Codex 三合一账号，就是这个逻辑——你不知道下个月谁最强，那就都买了。

## 今天没什么特别大的感慨

说实话，看完今天这期日报，我没有特别强烈的"哇这改变一切"的感觉。

Spotify 加了 Agent，CarPlay 也加了，DeepSeek 数学推理成本低到五百分之一……每一条单独看都挺大，但连着看，感觉更像是"2026年的日常"——Agent 在渗透所有入口，成本在持续下降，工具在继续爆炸，这已经是背景噪音了，不是新闻了。

![Raycast Glaze 一句话造软件](https://pbs.twimg.com/media/HKLL4GAaIAAz5u7.jpg)

唯一让我真的有点触动的是 CareMind 那条——[有人给外婆照护做了一个 Agent](https://t.me/aigc1024/20991)，不做诊断，不替代医生，只是陪着一个普通女儿撑过每天的照护。我妈年纪也大了，我在大理，她在老家，这条新闻我看了两遍。

有时候觉得，AI 做的最有价值的事，不是写代码，不是做数学证明，是这种——让一个普通人在很难的处境里，稍微轻松一点点。
