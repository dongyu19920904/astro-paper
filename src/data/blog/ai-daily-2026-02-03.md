今天下午刷推的时候看到 Claude Sonnet 5 泄露的消息，第一反应不是"哇好厉害"，而是赶紧打开微信看供应商群。果然，已经有人在问"新模型出来价格怎么算"了。卖账号这行就是这样，技术新闻对我来说首先是商业信号。

## Table of contents

## Vertex AI 日志泄露这事，我信七成

说实话，看到 `claude-sonnet-5@20260203` 这个模型 ID 的时候，我心里是有点激动的。日期就是明天，这要么是 Anthropic 故意放风，要么是真的有人手滑了。

我倾向于相信是真的。为什么？因为 Anthropic 这家公司一直挺低调的，不像 OpenAI 那样喜欢搞发布会造势。他们更喜欢"突然上线，然后让用户自己发现"这种风格。去年 Claude 3.5 Sonnet 就是这么搞的，我记得当时我还在闲鱼卖 API，早上起来发现模型列表多了个新的，供应商群里已经吵翻天了。

SWE-bench 80.9% 这个数字如果是真的，那确实是碾压级别的。我虽然不太懂这些 benchmark 具体怎么测的，但我知道一件事：我卖的 Claude 账号，用户反馈最多的就是"写代码真的好用"。如果新模型更强，那我的销量应该会涨。

## 说到卖账号，最近遇到个有意思的事

上周有个客户来问我，说他买了 Cursor Pro 换号器，但是用了两天就不想用了，问能不能退。我问他为什么，他说"感觉 AI 写的代码不如自己写的好"。

我当时愣了一下。这种反馈我还是第一次收到。

后来聊了几句才明白，他是个有十几年经验的老程序员，写代码已经形成了自己的风格和习惯。AI 生成的代码虽然能跑，但"看着不顺眼"。他说每次都要花时间改成自己喜欢的样子，还不如自己写。

我没给他退款（毕竟用了两天了），但这事让我想了挺久。AI 编程工具的目标用户到底是谁？是像我这样的菜鸟，还是像他那样的老手？

我觉得可能是前者。老手有自己的肌肉记忆，AI 反而是干扰。但对我这种写个 Python 脚本都要查半天语法的人来说，AI 就是救命稻草。

## OpenAI 额度翻倍这事，我有点酸

Sam Altman 说 Codex App 上线，付费用户额度翻倍两个月。

我酸的点在于：我卖的主要是 Claude 和 Cursor，OpenAI 的产品我进货不多。现在 OpenAI 搞这种促销，会不会把用户抢走？

不过转念一想，AI 工具这个市场还在快速增长，不是零和游戏。用户今天用 Codex，明天可能还是会来买 Claude。毕竟不同工具有不同的优势，很多人是多个一起用的。

而且说实话，我自己也想试试 Codex。虽然我是卖 Claude 账号的，但我对 OpenAI 的产品没有偏见。哪个好用用哪个，这是用户的逻辑，也应该是我的逻辑。

## 李想那条朋友圈，我看懂了一半

他点名的六款产品：Claude Code、豆包手机、Manus、OpenClaw、MoltBook、Chrome Gemini。

我用过的只有 Claude Code 和 Chrome Gemini。其他几个要么没听过，要么听过但没用过。

但他说的"GUI Agent"这个概念我是认同的。我之前帮客户解决问题的时候，经常遇到一个困境：用户想让 AI 帮他操作某个软件，但那个软件没有 API，AI 只能干瞪眼。

如果 AI 能直接看屏幕、模拟点击，那很多问题就迎刃而解了。这不就是我们想象中的"贾维斯"吗？

不过我也有担心：这种技术如果普及了，会不会有安全问题？AI 能看你的屏幕，能帮你点击，那它是不是也能看到你的密码、银行卡号？

我不是技术专家，这个问题我回答不了。但我觉得这是个值得关注的点。

## 今天的日报其实挺无聊的

除了 Claude Sonnet 5 泄露这条，其他新闻都是"又有个新工具发布了"、"又有个模型开源了"这种。

不是说这些不重要，而是看多了会麻木。每天都有新东西出来，但真正能改变我生活的，一年也就那么几个。

明天如果 Claude Sonnet 5 真的发布了，我会第一时间测试，然后更新我的产品线。如果没发布，那就继续等。

反正在大理，时间多的是。
