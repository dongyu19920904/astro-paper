今天看到千问接入阿里全家桶的新闻，第一反应不是"哇好方便"，而是——卧槽，这玩意儿要是能帮我自动处理订单就好了。

## Table of contents

作为一个在大理卖 AI 账号的人，我每天要处理的事情其实挺琐碎的：回复客户咨询、发货、处理售后、盯着供应商有没有跑路、看各种 AI 工具有没有更新……说白了，我也需要一个"AI 办事助手"，只不过它要会的不是点外卖，而是帮我盯着 Cursor 有没有又改规则、Claude 账号库存还剩多少、今天有几个客户在等发货。

## 千问这波操作，让我想起去年的一个坑

去年我在闲鱼卖 DeepSeek API 的时候，有段时间订单突然暴增。那会儿我一个人忙不过来，想着能不能搞个自动发货系统。结果折腾了半天，发现闲鱼根本不开放 API，只能手动一个个发。

那段时间我每天盯着手机，生怕漏掉哪个订单。有一次在咖啡馆发呆，手机突然震了十几下——十几个人同时下单。我当时就慌了，赶紧一个个复制粘贴账号密码，手指都按麻了。

现在看到千问能一句话点外卖、订机票，我就在想：要是它能接入闲鱼、接入我的供应商系统，帮我自动处理这些破事儿，那该多好。

## Claude Code 的更新，对我来说是好消息

今天另一条让我眼前一亮的新闻是 Claude Code 更新了 MCP 工具搜索功能。以前装了一堆 MCP 工具，每次检索都要占用大量上下文，效率低得要命。现在支持动态加载了，不用的工具不占上下文，这个改进太实用了。

我自己也用 Cursor 和 Claude Code 写点小工具（虽然写得很菜），最烦的就是上下文不够用。有时候写到一半，突然提示"上下文满了"，前面的对话全白费，得重新开一个会话。这次更新的 Tab 键补充信息功能也很贴心——以前 Claude 给的方案对了一半错了一半，只能选"同意"或"拒绝"，现在能直接告诉它哪里需要改，省了无数次重新对话的时间。

说实话，我对这些编程工具的技术细节其实不太懂，但作为一个普通用户，我能感受到哪些更新是真的有用，哪些是"看起来很酷但没啥卵用"。Claude Code 这次的更新，属于前者。

## 国产 GPU 的新闻，我看不太懂但感觉很重要

今天还有一条关于国产 GPU 的新闻，说有家成立不到一年的公司，两款芯片直接量产落地了。我对芯片这块儿完全是外行，但有一句话我记得很清楚："卡买回来了，然后呢？"

这句话说的是，国产 AI 芯片过去两年最大的问题不是"能不能造"，而是"能不能用"。参数跑分再漂亮，用户拿到手不会用、用不好，等于白搭。

我卖账号这一年多，也遇到过类似的问题。有些工具功能很强大，但用户体验极差，客户买了之后不会用，天天来问我怎么操作。最后我干脆写了一堆教程，手把手教他们怎么用。

所以我觉得，国产 GPU 要真正起来，不光要把芯片造出来，还得把生态做起来——让开发者愿意用、用得顺手。这家公司能量产落地，说明至少在"能用"这一步上迈出去了。具体效果怎么样，还得看后续反馈。

## Pixverse 的实时视频生成，让我想起一个疯狂的念头

今天看到 Pixverse 发布了 R1 实时视频世界模型，极限情况下能实时生成 1080P 高清视频，而且可以随时通过提示词介入修改后续内容。这个技术让我想起了我那个"疯狂梦想"——借 AI 实现长生不老。

我知道这听起来像民科，但我是认真的。我一直在关注 AI 在生命科学领域的进展，虽然我承认自己是小白，但直觉告诉我，AI 在这个领域的潜力远远没有被挖掘出来。

Pixverse 的实时视频生成技术，虽然现在主要用在游戏和影视内容上，但我在想：如果这种技术能用在医学影像分析上呢？比如实时生成病变组织的 3D 模型，帮助医生更直观地看到病灶的变化？

扯远了。回到现实，Pixverse 这个技术对我卖账号的生意没啥直接影响，但我觉得它代表了一个趋势——AI 正在从"生成静态内容"走向"实时交互"。这个趋势，可能会改变很多行业。

## 今天没什么特别的结论，就是随便聊聊

今天的日报看下来，没有什么让我特别震撼的新闻，但有几条让我想起了自己的一些经历和困惑。千问接入阿里生态，让我想起去年在闲鱼手动发货的痛苦；Claude Code 的更新，让我觉得这些工具正在变得越来越好用；国产 GPU 的新闻，让我想起"能用"比"能造"更重要；Pixverse 的实时视频生成，让我想起我那个疯狂的长生不老梦想。

说实话，我也不知道这些新闻对我的生意会有什么具体影响。可能千问以后会推出商家版，帮我自动处理订单？可能 Claude Code 会越来越好用，让我这种编程小白也能写出像样的工具？可能国产 GPU 会降低算力成本，让我进货更便宜？

我不知道。但我知道的是，AI 这个行业变化太快了，今天看起来不可能的事情，明天可能就实现了。所以我只能一边卖账号，一边观察，一边学习，一边等待。

在大理的咖啡馆里，敲下这些字的时候，窗外的苍山云雾缭绕。我突然想起去年刚来大理的时候，还在为每天几十块的收入焦虑。现在虽然月入稳定了几千，但新的焦虑又来了——这个行业会不会突然变天？我卖的这些账号，会不会哪天就没人要了？

不知道。但至少今天，我还在这里，还在卖账号，还在写博客，还在做梦。
